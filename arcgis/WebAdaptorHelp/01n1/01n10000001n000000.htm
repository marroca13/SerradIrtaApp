Getting started with ArcGIS Web Adaptor

You can configure ArcGIS Web Adaptor to work with ArcGIS Server or with Portal for ArcGIS. Configuring a single ArcGIS Web Adaptor application to work with both, though technically possible, is not supported. Once you decide how you'll be using ArcGIS Web Adaptor, see the sections below to get started.

Use ArcGIS Web Adaptor with ArcGIS Server

If you'll be using ArcGIS Web Adaptor solely with ArcGIS Server, see the following topics to get started:

  • About the ArcGIS Web Adaptor
  • An overview of implementing the Web Adaptor with ArcGIS Server

Use ArcGIS Web Adaptor with Portal for ArcGIS

ArcGIS Web Adaptor is a required component of Portal for ArcGIS. See the following topics to get started:

  • About the ArcGIS Web Adaptor
  • An overview of implementing the Web Adaptor with Portal for ArcGIS
