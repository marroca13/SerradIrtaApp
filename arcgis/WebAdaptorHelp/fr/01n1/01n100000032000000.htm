Questions, commentaires et informations

My Esri

My Esri est un site Web qui vous permet d'interagir avec Esri pour répondre à des questions, publier vos commentaires et obtenir des informations. Vous y trouverez également les questions fréquemment posées, des instructions sur les procédures à suivre, des téléchargements de logiciels, des conseils de dépannage, etc. Connectez-vous au site Web à l'aide de votre compte Esri pour interagir dès à présent avec Esri. Vous pouvez également contacter le service client ou votre distributeur local.

Aide sur le Web

Visitez les sites Web ArcGIS Desktop, ArcGIS Enterprise, et ArcGIS for Developers pour obtenir des informations actualisées sur les logiciels et les services ArcGIS. Ces précieuses ressources vous aideront à comprendre la technologie SIG.

Aide installée

L'Aide est installée en même temps que le logiciel ArcGIS. Sur les ordinateurs équipés d'un système d'exploitation Windows, les raccourcis vers l'Aide sont disponibles dans Démarrer > Tous les programmes > ArcGIS. Vous pouvez localiser les informations dont vous avez besoin en effectuant une recherche dans la table des matières, en recherchant des termes dans l'index ou en déterminant le nombre de fois où un terme apparaît dans une section.
