Installation de plusieurs instances d'ArcGIS Web Adaptor

Il est possible d'installer plusieurs instances d'ArcGIS Web Adaptor simultanément sur la même machine. L'installation peut également être répétée sur plusieurs machines. La procédure suivante vous guide dans ce processus.

Procédure d'installation de plusieurs Adaptateurs Web

Il est possible d'installer plusieurs adaptateurs Web sur le même ordinateur en même temps ou sur plusieurs ordinateurs. Pour installer plusieurs adaptateurs Web sur le même site Web (port), vous devez utiliser plusieurs noms différents. Par exemple, deux Adaptateurs Web nommés arcgis ne peuvent pas coexister sur le même site Web. Pour installer plusieurs adaptateurs Web portant le même nom, vous devez les installer sur des sites Web différents (ports).

  1. Accédez au dossier téléchargé contenant le fichier d'installation d'ArcGIS Web Adaptor (IIS) ou insérez le support ArcGIS Server dans le lecteur approprié pour lancer automatiquement l'interface de support.
  2. Suivez les mêmes instructions que pour l'installation du premier Adaptateur Web. Pour plus d'informations, reportez-vous à la rubrique Installation d'ArcGIS Web Adaptor.
Remarque :

Le programme d'installation détecte et prévient tout conflit avec le nom de l'Adaptateur Web. Si un message s'affiche indiquant que le répertoire virtuel existe déjà, spécifiez un nom différent pour l'Adaptateur Web ou sélectionnez un site Web différent.

Remarque :

Pour appliquer un Service Pack Web Adaptor, l'instance de base ne peut pas être supprimée.

Procédure d'installation de plusieurs Adaptateurs Web en mode silencieux

Il est possible d'installer plusieurs Adaptateurs Web en exécutant le programme d'installation à l'aide des paramètres de ligne de commande de Windows Installer. Reportez-vous à la rubrique Installation d'ArcGIS Web Adaptor en mode silencieux pour des instructions complètes sur l'utilisation des paramètres de ligne de commande. Aucun paramètre de ligne de commande supplémentaire n'est nécessaire pour installer plusieurs Adaptateurs Web.

Procédure de désinstallation de plusieurs Adaptateurs Web

Chaque Adaptateur Web doit être désinstallée séparément. Pour obtenir des instructions complètes, reportez-vous à la rubrique Désinstallation d'ArcGIS Web Adaptor.
