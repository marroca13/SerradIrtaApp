Configuration de la langue d'affichage d'ArcGIS Web Adaptor

Le programme d'installation d'ArcGIS Web Adaptor inclut des versions prélocalisées de Web Adaptor pour différentes langues. Ces langues sont les suivantes :

  • Arabe
  • Chinois simplifié
  • Français
  • Allemand
  • Italien
  • Japonais
  • Portugais (Brésil)
  • Russe
  • Espagnol
  • Turc

Une fois l'installation terminée, la page de configuration de Web Adaptor s'ouvre automatiquement. Si votre navigateur est déjà configuré pour afficher l'une des langues ci-dessus, l'Adaptateur Web s'ouvre automatiquement dans cette langue.

Pour afficher Web Adaptor dans une langue différente, configurez votre navigateur ou votre système selon la procédure suivante.

Utilisateurs de Firefox

Des versions localisées de Firefox sont disponibles. L'installation d'une version localisée de Firefox et l'ouverture de Web Adaptor l'affichent automatiquement dans la langue du navigateur.

Si vous souhaitez utiliser Firefox et afficher l'adaptateur Web dans une autre langue que celle du navigateur, vous devez utiliser Firefox 10 ou une version ultérieure et configurer les éléments suivants :

  1. Sélectionnez Outils > Options.
  2. Sous l'onglet Contenu, sous Langues, cliquez sur Choisir.
  3. Sélectionnez la langue à afficher et placez-la en tête de la liste des langues.

Utilisateurs de Chrome

Si vous utilisez Google Chrome et que vous souhaitez modifier la langue d'affichage de l'Adaptateur Web, configurez les éléments suivants :

  1. Sélectionnez Personnaliser > Options > Sous le capot.
  2. Dans la section de contenu Web, cliquez sur les options de langue et de correction automatique.
  3. Sélectionnez la langue à afficher et cliquez sur Afficher Google Chrome dans cette langue.

Utilisateurs d'Internet Explorer

Si vous utilisez Internet Explorer et que vous souhaitez modifier la langue d'affichage de l'Adaptateur Web, configurez les éléments suivants :

  1. Sélectionnez Outils > Options Internet.
  2. Sous l'onglet Général, sous Apparence, cliquez sur le bouton Langues.
  3. Sélectionnez la langue à afficher et placez-la en tête de la liste des langues. Cliquez sur OK deux fois pour accepter la modification.
