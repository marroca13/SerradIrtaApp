Activation automatique des composants IIS

Pour pouvoir installer ArcGIS Web Adaptor, certains composants Microsoft Internet Information Services (IIS) 7.0, 7.5, 8.0 ou 8.5 sont requis. Si vous avez déjà activé IIS 7.0, 7.5, 8.0 ou 8.5, mais que les composants IIS requis sont manquants, l'installation affiche un message indiquant que certains composants IIS sont manquants.

Vous avez la possibilité d'autoriser l'installation de l'adaptateur Web à activer automatiquement les composants IIS requis. Vous pouvez également activer les composants vous-même. Le programme d'installation ne se poursuit pas si les composants IIS requis ne sont pas activés.

Suivez les instructions présentées dans les sections ci-après pour activer automatiquement ou manuellement les composants IIS manquants.

Activation automatique des composants IIS manquants

  1. Passez en revue les composants IIS manquants dans la boîte de dialogue Vérification des conditions requises relatives à ISS. Il peut s'avérer utile de consigner les composants manquants.
  2. Pour activer automatiquement les composants manquants, cliquez sur J'accepte.

Les composants sont activés et vous pouvez poursuivre le processus d'installation.

Activation manuelle des composants IIS manquants

Pour activer manuellement les composants IIS 7.0, 7.5, 8.0 ou 8.5 manquants, cliquez sur Annuler. Pour savoir comment activer manuellement les composants IIS manquants par vous-même, consultez les rubriques ci-après.

  • Activation d'IIS et des composants IIS requis sous Windows Server 2012
  • Activation d'IIS et des composants IIS requis sous Windows Server 2008/2008 R2
  • Activation d'IIS et des composants IIS requis sous Windows 8/8.1
  • Activation d'IIS et des composants IIS requis sous Windows 7

Une fois tous les composants requis activés, relancez le processus d'installation de Web Adaptor en double-cliquant sur le fichier Setup.exe, contenu dans le dossier d'installation d'ArcGIS Web Adaptor (IIS).
