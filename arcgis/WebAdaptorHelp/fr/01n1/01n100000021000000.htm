Vue d'ensemble de l'implémentation d'ArcGIS Web Adaptor avec Portal for ArcGIS

Voici une vue d'ensemble de la procédure à suivre pour implémenter ArcGIS Web Adaptor avec Portal for ArcGIS :

Etapes :
  1. Planifiez la façon dont vous allez utiliser ArcGIS Web Adaptor avec Portal for ArcGIS.
  2. Vérifiez que le système respecte la configuration système requise.
  3. Activer HTTPS sur votre serveur Web.
  4. Installez ArcGIS Web Adaptor.
  5. Configurez ArcGIS Web Adaptor pour l'utiliser avec Portal for ArcGIS.
