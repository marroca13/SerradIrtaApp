Vue d'ensemble de l'implémentation d'ArcGIS Web Adaptor avec ArcGIS Server

Voici une vue d'ensemble de la procédure à suivre pour implémenter Web Adaptor avec ArcGIS Server :

  1. Planifiez la configuration de votre site ArcGIS Server.
  2. Vérifiez que le système respecte la configuration système requise.
  3. Installez ArcGIS Web Adaptor.
  4. Configurez ArcGIS Web Adaptor pour l'utiliser avec ArcGIS Server.
