Configuration système requise pour ArcGIS Web Adaptor 10.6

Les configurations système et matérielle requises pour exécuter ArcGIS Web Adaptor sont répertoriées ci-dessous. Pour connaître la configuration mise à jour et les exigences supplémentaires, consultez la page Configuration système requise disponible sur le web. Pour plus d'informations sur les versions antérieures, consultez le support technique Esri.

Nous vous recommandons de prendre connaissance de la notice correspondante pour déterminer si vos composants matériels et logiciels sont toujours compatibles avec la version 10.6.

Les mises à jour ou Service Packs antérieurs ou futurs de ces systèmes d'exploitation sont pris en charge, sauf indication contraire. La version et les mises à jour du système d'exploitation doivent également être prises en charge par le fournisseur du système d'exploitation.

Remarque :

Windows 7, 8.1 et 10 sont pris en charge pour les tests de base et le développement des applications uniquement. Ils ne sont pas recommandés pour un déploiement dans un environnement de production.

Nombre maximal d'instances d'installation

Vous pouvez installer au maximum 21 instances d'ArcGIS Web Adaptor sur une seule machine. Si plus d'instances sont nécessaires, vous devez les installer sur une autre machine.

Si des versions antérieures sont installées sur la machine, il n'est pas nécessaire de les désinstaller. Par exemple, 5 instances d'ArcGIS Web Adaptor 10.4 et 21 instances d'ArcGIS Web Adaptor 10.6 peuvent être installées sur la même machine. La valeur maximale s'applique uniquement aux instances de la même version logicielle.

Prise en charge de Microsoft .NET Framework 4.5

Pour installer ArcGIS Web Adaptor, vous devez disposer de Microsoft .NET Framework 4.5 ou une version ultérieure. Si votre système d'exploitation inclut Microsoft .NET Framework 4.5, vérifiez qu'il est activé avec les fonctionnalités de Windows. Si votre système d'exploitation n'inclut pas Microsoft .NET Framework 4.5, vous pouvez le télécharger à partir du support d'installation et l'activer avec les fonctionnalités de Windows.

Composants requis pour Microsoft IIS

ArcGIS Web Adaptor prend en charge IIS sous Windows 7, 8.1, 10, Server 2008 R2, Server 2012 et 2012 R2. Ceci correspond aux versions IIS 7.0, 7.5, 8.0, 8.5 et 10. IIS doit être activé, ainsi que certains composants spécifiques d'IIS 7.0, 7.5, 8.0, 8.5 et 10. Le programme d'installation ne se poursuit pas si IIS n'est pas détecté et que les composants IIS spécifiques ne sont pas activés.

Si vous avez déjà installé IIS 7.0, 7.5, 8.0 ou 8.5 mais que des composants IIS requis manquent, le programme d'installation affiche la boîte de dialogue Vérification des conditions requises relatives à IIS. Cette boîte de dialogue permet d'autoriser l'installation à activer automatiquement tous les composants IIS requis manquants. Pour ce faire, cliquez sur l'icône J'accepte. Pour en savoir plus, reportez-vous à la rubrique Activation automatique des composants IIS.

Remarque :

Si vous prévoyez d'effectuer une installation en mode silencieux d'ArcGIS Web Adaptor, tous les composants IIS requis doivent être activés manuellement. En effet, le programme d'installation n'active pas automatiquement les composants IIS requis lorsque vous effectuez une installation en mode silencieux.

Pour savoir comment activer IIS sur votre système d'exploitation et, facultativement, activer les composants IIS 7.0, 7.5, 8.0 ou 8.5 manquants par vous-même, consultez les rubriques suivantes :
