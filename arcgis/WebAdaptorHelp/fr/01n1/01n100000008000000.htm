Installation d'ArcGIS Web Adaptor

ArcGIS Web Adaptor est fourni sous la forme d'un programme d'installation. Il est exécuté au moyen d'un utilitaire nommé Setup.exe. Vous pouvez installer ArcGIS Web Adaptor seul en suivant les étapes ci-dessous ou procéder à une installation en mode silencieux à l'aide de paramètres de ligne de commande de Windows Installer.

Comportement et exigences requis pour l'installation

La version d'ArcGIS Web Adaptor doit toujours correspondre à celle de son site ArcGIS Server inscrit.

Vous pouvez installer ArcGIS Web Adaptor avec d’autres versions. Vous n’êtes pas obligé de désinstaller ArcGIS Web Adaptor pour installer ArcGIS Web Adaptor si ArcGIS Web Adaptor n’est pas une mise à jour de versions antérieures.

Chaque instance d’ArcGIS Web Adaptor doit avoir un nom unique. Deux instances du même nom ne peuvent pas coexister sur le même serveur Web. En présence d'un conflit d'attribution de nom pendant l'installation, un message d'avertissement apparaît. Vous devez désinstaller la version antérieure du même nom pour résoudre le conflit. Vous pouvez également modifier le nom d'un des adaptateurs Web pour poursuivre l'installation.

Mettre à niveau une version antérieure d'ArcGIS Web Adaptor

Pour mettre à niveau une version précédente d’ArcGIS Web Adaptor vers la version 10.6, vous devez désinstaller la version précédente et installer la version 10.6. Pendant l’installation de la version 10.6, vous devez indiquer le nom de l'adaptateur Web précédent. Ainsi, l'URL de l'adaptateur Web sera identique à celle de la version précédente.

En présence d'un conflit d'attribution de nom pendant l'installation de la version 10.6, un message d'avertissement apparaît. Vous devez désinstaller la version antérieure du même nom pour procéder à la mise à niveau.

Après avoir installé la version 10.6, vous devez configurer ArcGIS Web Adaptor avec son site ArcGIS Server correspondant.

Héritage :

Dans les versions 10.2 et antérieures, il suffisait d'exécuter le fichier d'installation pour mettre à niveau toutes les versions existantes des adaptateurs Web.

Avant d'installer ArcGIS Web Adaptor

  1. Vérifiez que votre système respecte la configuration requise.
  2. Connectez-vous en tant qu'utilisateur possédant des droits d'administration.
  3. Fermez toutes les applications.
  4. Accédez au dossier téléchargé contenant le programme d'installation d'ArcGIS Web Adaptor. Vous pouvez également insérer le support de ArcGIS Server dans le lecteur approprié pour démarrer automatiquement le programme d'installation.

Vous êtes maintenant prêt à installer l'adaptateur Web.

Remarque :

Il est impossible de configurer ArcGIS Web Adaptor de manière à ce que l'URL comprenne des niveaux supplémentaires et cette opération peut entraîner des problèmes avec les applications clientes.

Procédure d'installation d'ArcGIS Web Adaptor

  1. Le programme d'installation d'ArcGIS Web Adaptor doit démarrer automatiquement une fois le téléchargement terminé. Si le programme d’installation ne démarre pas automatiquement, accédez au dossier dans lequel vous avez téléchargé les fichiers d'installation et double-cliquez sur Setup.exe.

  2. Lisez les termes et conditions du contrat principal. Vous devez accepter les termes du contrat pour continuer.

  3. Choisissez un site Web pour l'adaptateur Web. Les sites Web disponibles sont répertoriés sous la forme <nom du site Web (port)>. Si la boîte de dialogue Sélectionnez un site Web ne s'affiche pas, cela signifie que vous disposez d'un seul site Web. Si un seul site Web est localisé sur votre machine, ArcGIS Web Adaptor est automatiquement placé dessus sans que la boîte de dialogue Sélectionnez un site Web s'affiche.

  4. Entrez le nom de l'adaptateur Web. La valeur par défaut est arcgis. Le nom de cette instance ne peut pas contenir d'espaces.

    Remarque :

    Un message s'affiche si un répertoire virtuel portant le même nom que l'adaptateur Web existe déjà sur le site Web sélectionné. Cliquez sur OK pour fermer le message et donnez un autre nom à l'adaptateur Web.

  5. Pour terminer l'installation, suivez les instructions à l'écran.

Une fois l'installation terminée, la page de configuration de l'adaptateur Web apparaît. Vous êtes maintenant prêt à Configurer ArcGIS Web Adaptor pour l’utiliser avec ArcGIS Server.
