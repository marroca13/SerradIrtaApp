Activation d'IIS et des composants IIS requis sous Windows Server 2012/2012 R2

ArcGIS Web Adaptor nécessite qu'IIS et que certains composants IIS spécifiques soient activés sous Windows Server 2012/2012 R2. Le programme d'installation ne se poursuit pas si IIS n'est pas détecté et que les composants IIS spécifiques ne sont pas activés.

Si vous avez déjà activé IIS, mais que les composants IIS requis manquent, l'installation affiche un message indiquant que certains composants IIS sont manquants. Vous avez la possibilité d'autoriser l'installation à activer automatiquement les composants IIS. Cependant, selon les règles de sécurité de votre organisation, il peut s'avérer nécessaire d'activer manuellement les composants IIS requis, comme décrit ci-après.

Pour savoir comment activer IIS et les composants IIS requis sous Windows Server 2012/2012 R2, consultez les instructions ci-après.

  1. Ouvrez Gestionnaire de serveur et cliquez sur Gérer > Ajouter des rôles et fonctionnalités. Cliquez sur Suivant.
  2. Sélectionnez Installation basée sur un rôle ou une fonctionnalité et cliquez sur Suivant.
  3. Sélectionnez le serveur approprié. Le serveur local est sélectionné par défaut. Cliquez sur Suivant.
  4. Activez Serveur Web (IIS) et cliquez sur Suivant.
  5. Aucune fonctionnalité supplémentaire n'étant nécessaire à l'installation de l'adaptateur Web, cliquez sur Suivant.
  6. Dans la boîte de dialogue Rôle Serveur Web (IIS), cliquez sur Suivant.
  7. Dans la boîte de dialogue Sélectionner les services de rôle, vérifiez que les composants de serveur Web ci-dessous sont activés. Cliquez sur Suivant.
  8. Vérifiez l'exactitude de vos paramètres et cliquez sur Installer.
  9. Au terme de l'installation, cliquez sur Fermer pour quitter l'assistant.

Composants IIS requis

Les composants IIS répertoriés ci-après correspondent à la configuration minimale requise pour exécuter Web Adaptor. Si d'autres composants IIS sont activés, ils n'ont pas besoin d'être supprimés.

    • Fonctionnalités HTTP communes
      • Document par défaut
      • Contenu statique
    • Sécurité
      • Filtrage des demandes
      • Authentification de base
      • Authentification Windows
    • Développement d'applications
      • .NET Extensibility 4.5
      • Extensibilité .NET
      • ASP.NET 4.5
      • ASP.NET
      • Extensions ISAPI
      • Filtres ISAPI
  • Outils de gestion
    • Console de gestion IIS
    • Compatibilité avec la gestion IIS 6
      • Compatibilité avec la métabase IIS 6
    • Scripts et outils de gestion IIS
    • Service de gestion
