Activation d'IIS et des composants IIS requis sous Windows Server 2008/2008 R2

ArcGIS Web Adaptor nécessite qu'IIS et que certains composants IIS spécifiques soient activés sous Windows Server 2008/2008 R2. Le programme d'installation ne se poursuit pas si IIS n'est pas détecté et que les composants IIS spécifiques ne sont pas activés.

Si vous avez déjà activé IIS, mais que les composants IIS requis manquent, l'installation affiche un message indiquant que certains composants IIS sont manquants. Vous avez la possibilité d'autoriser l'installation à activer automatiquement les composants IIS. Cependant, selon les règles de sécurité de votre organisation, il peut s'avérer nécessaire d'activer manuellement les composants IIS requis, comme décrit ci-après.

Pour savoir comment activer IIS et les composants IIS requis sous Windows Server 2008/2008 R2, consultez les instructions ci-après.

  1. Cliquez sur Démarrer > Outils administratifs > Gestionnaire de serveurs.

  2. Dans le volet de gauche de la boîte de dialogue Gestionnaire de serveurs, cliquez sur Rôles.
    • Si IIS n'a pas été activé, cliquez sur Ajouter des rôles dans le panneau Récapitulatif des rôles. Cliquez sur Suivant et activez Serveur Web (IIS) dans la liste. Cliquez ensuite sur Suivant, puis sélectionnez Services de rôle dans le panneau gauche.
    • Si IIS est déjà installé mais que les composants requis ne sont pas tous activés, cliquez sur Ajouter des services de rôle dans le panneauServeur Web (IIS) à droite.

  3. Dans la boîte de dialogue Sélectionner les services de rôle, vérifiez que les composants de serveur Web ci-dessous sont activés.

  4. Après avoir activé les composants IIS requis, cliquez sur Suivant > Installer.

Composants IIS requis

Les composants IIS répertoriés ci-après correspondent à la configuration minimale requise pour exécuter Web Adaptor. Si d'autres composants IIS sont activés, ils n'ont pas besoin d'être supprimés.

  • Serveur Web
    • Fonctionnalités HTTP communes
      • Contenu statique
      • Document par défaut
    • Développement d'applications
      • ASP.NET
      • Extensibilité .NET
      • Extensions ISAPI
      • Filtres ISAPI
    • Sécurité
      • Authentification de base
      • Authentification Windows
      • Filtrage des demandes
  • Outils de gestion
    • Console de gestion IIS
    • Scripts et outils de gestion IIS
    • Service de gestion
    • Compatibilité avec la gestion IIS 6
      • Compatibilité avec la métabase IIS 6
