Activation d'IIS et des composants IIS requis sous Windows 8/8.1

ArcGIS Web Adaptor nécessite qu'IIS et que certains composants IIS spécifiques soient activés sous Windows 8/8.1. Le programme d'installation ne se poursuit pas si IIS n'est pas détecté et que les composants IIS spécifiques ne sont pas activés.

Si vous avez déjà activé IIS, mais que les composants IIS requis manquent, l'installation affiche un message indiquant que certains composants IIS sont manquants. Vous avez la possibilité d'autoriser l'installation à activer automatiquement les composants IIS. Cependant, selon les règles de sécurité de votre organisation, il peut s'avérer nécessaire d'activer manuellement les composants IIS requis, comme décrit ci-après.

Pour savoir comment activer IIS et les composants IIS requis sous Windows 8/8.1, consultez les instructions ci-après.

  1. Ouvrez Panneau de configuration, puis cliquez sur Programmes et fonctionnalités > Activer ou désactiver des fonctionnalités Windows.
  2. Activez Internet Information Services.
  3. Développez la fonctionnalité Internet Information Services et vérifiez que les composants de serveur Web ci-dessous sont activés.
  4. Cliquez sur OK.
