Désinscrire ArcGIS Web Adaptor d'ArcGIS Server

Pour modifier ArcGIS Server afin qu'il utilise une autre instance d'ArcGIS Web Adaptor, pour modifier le nom de l'adaptateur Web ou de la machine hébergeant ArcGIS Web Adaptor, par exemple, vous devez tout d'abord désinscrire ArcGIS Web Adaptor de votre site ArcGIS Server.

Pour désinscrire ArcGIS Web Adaptor de votre site ArcGIS Server, vous devez utiliser le Répertoire d'administrateur d'ArcGIS Server. Vous ne pouvez procéder à une désinscription que via le Répertoire d'administrateur d'ArcGIS Server. La fonctionnalité de désinscription n'est pas disponible à partir de la page de configuration d'ArcGIS Web Adaptor.

Pour désinscrire ArcGIS Web Adaptor, procédez comme suit.

Etapes :
  1. Ouvrez le répertoire d'administrateur d'ArcGIS Server et connectez-vous en tant qu'administrateur de votre site. L'URL est au format https://gisserver.domain.com:6443/arcgis/admin.
  2. Cliquez sur Système > Adaptateurs Web.
  3. Sur la page Adaptateurs Web, cliquez sur la chaîne représentant ArcGIS Web Adaptor.
  4. Sur la page Adaptateur Web, vérifiez que l'URL répertoriée désigne l'adaptateur Web que vous souhaitez inscrire. Cliquez sur Désinscrire.
  5. Sur la page Désinscrire l'adaptateur Web, cliquez sur Désinscrire l'adaptateur Web. ArcGIS Web Adaptor est désinscrit de votre site ArcGIS Server.
  6. Configurez la nouvelle instance d'ArcGIS Web Adaptor auprès de votre site ArcGIS Server. Pour en savoir plus, reportez-vous à la rubrique Configurer ArcGIS Web Adaptor.
Remarque :

Si vous avez l'intention de supprimer entièrement l'instance précédente d'ArcGIS Web Adaptor de votre site ArcGIS Server, nous vous recommandons de désinstaller ArcGIS Web Adaptor après avoir suivi les étapes ci-dessus. La désinstallation entraîne la suppression de tous les détails de configuration d'ArcGIS Web Adaptor du site et empêche les membres et clients d'accéder au site via l'URL de l'instance d'ArcGIS Web Adaptor désinscrite. Pour obtenir des instructions complètes, reportez-vous à la rubrique Désinstaller ArcGIS Web Adaptor.
