Aufheben der Registrierung von ArcGIS Web Adaptor bei ArcGIS for Server

Wenn Sie ArcGIS for Server für die Verwendung einer anderen Version von ArcGIS Web Adaptor ändern möchten, d. h. beispielsweise den Namen des Web Adaptor oder den Namen des Computers, der ArcGIS Web Adaptor hostet, ändern, müssen Sie zuerst die Registrierung von ArcGIS Web Adaptor bei Ihrer ArcGIS-Server-Site aufheben.

Zum Aufheben der Registrierung von ArcGIS Web Adaptor bei Ihrer ArcGIS-Server-Site verwenden Sie das ArcGIS-Server-Administratorverzeichnis. Die Aufhebung der Registrierung kann nur über das ArcGIS-Server-Administratorverzeichnis erfolgen; die Funktion zum Aufheben der Registrierung ist auf der Konfigurationsseite von ArcGIS Web Adaptor verfügbar.

Führen Sie die folgenden Schritte aus, um die Registrierung von ArcGIS Web Adaptor aufzuheben.

Schritte:
  1. Öffnen Sie das ArcGIS-Server-Administratorverzeichnis, und melden Sie sich als Administrator Ihrer Site an. Die URL hat das Format https://gisserver.domain.com:6443/arcgis/admin.
  2. Klicken Sie auf System > Web Adaptor.
  3. Klicken Sie auf der Seite Web Adaptor auf die Zeichenfolge, die ArcGIS Web Adaptor darstellt.
  4. Stellen Sie auf der Seite Web Adaptor sicher, dass die aufgeführte URL auf den Web Adaptor verweist, dessen Registrierung Sie aufheben möchten. Klicken Sie auf Registrierung entfernen.
  5. Klicken Sie auf der Seite Registrierung von Web Adaptor aufheben auf Registrierung von Web Adaptor aufheben. Die Registrierung von ArcGIS Web Adaptor bei Ihrer ArcGIS-Server-Site wurde aufgehoben.
  6. Konfigurieren Sie die neue Version von ArcGIS Web Adaptor bei Ihrer ArcGIS-Server-Site. Vollständige Anweisungen finden Sie unter Konfigurieren von ArcGIS Web Adaptor.
Hinweis:

Wenn Sie die vorherige Version von ArcGIS Web Adaptor vollständig von der ArcGIS-Server-Site entfernen möchten, empfiehlt es sich, ArcGIS Web Adaptor zu deinstallieren, nachdem Sie die Schritte oben ausgeführt haben. Beim Deinstallieren werden sämtliche Konfigurationsdetails von ArcGIS Web Adaptor von der Site gelöscht und der Zugriff auf die Site über die nicht registrierte ArcGIS Web Adaptor-URL wird für Mitglieder und Clients gesperrt. Vollständige Anweisungen finden Sie unter Deinstallieren von ArcGIS Web Adaptor.
