Systemanforderungen für ArcGIS Web Adaptor 10.6

Im Folgenden werden die für die Ausführung von ArcGIS Web Adaptor erforderlichen System- und Hardwareanforderungen aufgeführt. Zusätzliche und aktualisierte Anforderungen sind online unter Systemanforderungen verfügbar. Weitere Informationen zu früheren Versionen finden Sie unter Esri Support.

Es empfiehlt sich, die Deprecation Notes zu überprüfen, um zu ermitteln, ob Ihre Hardware- und Softwarekomponenten weiterhin mit Version 10.6 kompatibel sind.

Maximale Anzahl von Installationsinstanzen

Sie können maximal 21 Instanzen von ArcGIS Web Adaptor auf einem Computer installieren. Wenn mehr Instanzen erforderlich sind, müssen diese auf einem separaten Computer installiert werden.

Wenn Sie frühere Versionen auf dem Computer installiert haben, müssen diese nicht deinstalliert werden. Sie können beispielsweise 5 Instanzen von ArcGIS Web Adaptor 10.4 und 21 Instanzen von ArcGIS Web Adaptor 10.6 auf demselben Computer installieren. Der Maximalwert gilt nur für Instanzen derselben Softwareversion.

Unterstützung für Microsoft .NET Framework 4.5

Für die Installation von ArcGIS Web Adaptor ist Microsoft .NET Framework 4.5 oder höher erforderlich. Wenn Ihr Betriebssystem Microsoft .NET Framework 4.5 enthält, stellen Sie mit Windows-Features sicher, dass es aktiviert ist. Wenn Ihr Betriebssystem Microsoft .NET Framework 4.5 nicht enthält, können Sie es von den Installationsmedien herunterladen und mit Windows-Features aktivieren.

Erforderliche Komponenten für Microsoft IIS

ArcGIS Web Adaptor unterstützt IIS unter Windows 7, 8.1, 10, Server 2008 R2, Server 2012 und 2012 R2. Dies entspricht den IIS-Versionen 7.0, 7.5, 8.0, 8.5 und 10. IIS muss zusammen mit bestimmten IIS 7.0-, 7.5-, 8.0-, 8.5- und 10-Komponenten aktiviert werden. Wenn IIS nicht gefunden wird und bestimmte IIS-Komponenten nicht aktiviert werden, wird die Installation nicht fortgesetzt.

Wenn IIS 7.0, 7.5, 8.0 oder 8.5 bereits installiert ist, die erforderlichen IIS-Komponenten jedoch fehlen, wird bei der Installation das Dialogfeld Überprüfen der IIS-Anforderungen angezeigt. Dies gibt Ihnen die Möglichkeit, die fehlenden IIS-Komponenten automatisch zu installieren. Klicken Sie dazu auf Ich stimme zu. Weitere Informationen finden Sie unter Automatisches Aktivieren von IIS-Komponenten.

Hinweis:

Bei der automatischen Installation von ArcGIS Web Adaptor müssen alle erforderlichen IIS-Komponenten manuell aktiviert werden. Wenn Sie eine automatische Installation durchführen, werden fehlende IIS-Komponenten beim Setup nicht automatisch aktiviert.

Informationen zur Aktivierung von IIS auf Ihrem Betriebssystem und der optionalen manuellen Aktivierung fehlender IIS 7.0-, 7.5-, 8.0- oder 8.5-Komponenten finden Sie in den folgenden Themen:

  • Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows 8.1
