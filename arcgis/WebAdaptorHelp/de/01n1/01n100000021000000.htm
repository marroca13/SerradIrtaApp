Überblick über die Implementierung von ArcGIS Web Adaptor mit Portal for ArcGIS

Nachfolgend finden Sie einen Überblick über die Schritte, die zur Implementierung von ArcGIS Web Adaptor mit Portal for ArcGIS ausgeführt werden müssen:

Schritte:
  1. Planen Sie, wie Sie ArcGIS Web Adaptor mit Portal for ArcGISverwenden..
  2. Überprüfen Sie, ob Ihr System die minimalen Systemvoraussetzungen erfüllt.
  3. Aktivieren Sie HTTPS auf dem Webserver.
  4. Installieren Sie ArcGIS Web Adaptor.
  5. Konfigurieren Sie ArcGIS Web Adaptor für die Verwendung mit Portal for ArcGIS.
