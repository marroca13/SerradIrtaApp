Installieren mehrerer Instanzen von ArcGIS Web Adaptor

Auf einem Computer können mehrere Instanzen von ArcGIS Web Adaptor gleichzeitig installiert werden. Die Installation kann auch auf getrennten Computern wiederholt ausgeführt werden. Die Schritte unten führen Sie durch die Installation.

So installieren Sie mehrere Web Adaptor-Instanzen

Auf einem Computer können mehrere Web Adaptor-Instanzen gleichzeitig installiert werden oder die Installation kann über mehrere getrennte Computer verteilt werden. Wenn Sie mehrere Web Adaptor-Instanzen auf derselben Website (Port) installieren möchten, müssen diese verschiedene Namen verwenden. Beispielsweise können zwei Web Adaptor-Instanzen mit dem Namen arcgis nicht auf derselben Website vorhanden sein. Wenn Sie mehrere Web Adaptor-Instanzen mit demselben Namen installieren möchten, müssen diese auf getrennten Websites (Ports) installiert werden.

  1. Navigieren Sie zu dem heruntergeladenen Ordner, der das Installationsprogramm für ArcGIS Web Adaptor (IIS) enthält, oder legen Sie die ArcGIS Server-Medien in das entsprechende Laufwerk ein, um das Medien-Front-End automatisch zu starten.
  2. Führen Sie die gleichen Schritte wie bei der ersten Installation von Web Adaptor aus. Weitere Informationen finden Sie unter Installieren von ArcGIS Web Adaptor.
Hinweis:

Bei der Installation werden Konflikte mit dem Web Adaptor-Namen erkannt und verhindert. Wenn eine Meldung mit dem Hinweis angezeigt wird, dass das virtuelle Verzeichnis bereits vorhanden ist, geben Sie einen anderen Namen für den Web Adaptor an, oder wählen Sie eine andere Website aus.

Hinweis:

Die Basisinstanz kann nicht entfernt werden, um ein Service Pack für Web Adaptor anzuwenden.

So installieren Sie mehrere Web Adaptor-Instanzen automatisch

Es können mehrere Web Adaptor-Instanzen installiert werden, indem das Installationsprogramm mit Befehlszeilenparametern für Windows Installer ausgeführt wird. Vollständige Anweisungen zum Verwenden der Befehlszeilenparamater finden Sie unter Automatisches Installieren von ArcGIS Web Adaptor. Zum Installieren mehrerer Web Adaptor-Instanzen sind keine zusätzlichen Befehlszeilenparameter erforderlich.

So deinstallieren Sie mehrere Web Adaptor-Instanzen

Jeder Web Adaptor muss einzeln deinstalliert werden. Vollständige Anweisungen finden Sie unter Deinstallieren von ArcGIS Web Adaptor.
