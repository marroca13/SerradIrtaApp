Installieren von ArcGIS Web Adaptor

ArcGIS Web Adaptor wird als Installationsprogramm bereitgestellt und über ein Dienstprogramm namens Setup.exe ausgeführt. Sie können ArcGIS Web Adaptor allein installieren, indem Sie die hier angegebenen Schritte ausführen, oder Sie können die Befehlszeilenparameter von Windows Installer für die automatische Installation verwenden.

Installationsvoraussetzungen

Die Version von ArcGIS Web Adaptor muss immer mit der verwendeten Portal for ArcGIS-Version übereinstimmen.

ArcGIS Web Adaptor kann parallel mit anderen Versionen installiert werden. Für die Installation von ArcGIS Web Adaptor muss ArcGIS Web Adaptor nicht deinstalliert werden, und ArcGIS Web Adaptor ist kein Upgrade einer früheren Version.

Jeder ArcGIS Web Adaptor muss einen eindeutigen Namen haben; es können keine zwei Web Adaptor mit dem gleichen Namen auf einem Webserver installiert werden. Wenn ein Namenskonflikt während der Installation erkannt wird, wird eine Warnmeldung angezeigt. Sie müssen die frühere Version mit dem gleichen Namen deinstallieren, um den Konflikt zu lösen. Alternativ können Sie den Namen eines Web Adaptors ändern, um mit der Installation fortzufahren.

Upgrade einer früheren Version von ArcGIS Web Adaptor

Um ein Upgrade einer früheren ArcGIS Web Adaptor-Version auf 10.6 durchzuführen, müssen Sie die frühere Version deinstallieren und das 10.6-Setup installieren. Geben Sie bei der 10.6-Installation den Namen der früheren Web Adaptor-Version an. Dadurch wird sichergestellt, dass die Web Adaptor-URL mit der der früheren Version identisch ist.

Wenn ein Namenskonflikt während der Installation von 10.6 erkannt wird, wird eine Warnmeldung angezeigt. Sie müssen die frühere Version mit dem gleichen Namen deinstallieren, um das Upgrade durchzuführen.

Nachdem Sie 10.6 installiert haben, müssen Sie ArcGIS Web Adaptor mit Portal for ArcGIS konfigurieren.

Ältere Versionen:

In Versionen bis 10.2 wurde für alle vorhandenen Web Adaptors durch die Ausführung des Setup-Programms ein Upgrade auf die neueste Version durchgeführt.

Vor der Installation von ArcGIS Web Adaptor

Schritte:
  1. Überprüfen Sie, ob das System alle Systemanforderungen erfüllt. Sie können nur einen Web Adaptor mit Portal for ArcGIS konfigurieren.
  2. Melden Sie sich als Benutzer mit Administratorberechtigungen an.
  3. Stellen Sie sicher, dass eine Website auf Port 80 ausgeführt wird und HTTPS auf Port 443 aktiviert ist. Dadurch soll die Vielzahl an verschlüsselten und unverschlüsselten Aufrufen an Portal for ArcGIS einbezogen werden. ArcGIS Web Adaptor kann nicht mit Portal for ArcGIS konfiguriert werden, wenn es auf einer Website installiert ist, die auf einem anderen Port als Port 80 und 443 ausgeführt wird. Anweisungen zum Erstellen einer Website finden Sie in der Produktdokumentation zum IIS. Weitere Informationen zum Einrichten und Verwenden von HTTPS finden Sie unter Aktivieren von HTTPS auf dem Webserver.
  4. Schließen Sie alle Anwendungen.
  5. Navigieren Sie zu dem heruntergeladenen Ordner mit dem ArcGIS Web Adaptor-Setup. Sie können auch das Portal for ArcGIS-Medium in das entsprechende Laufwerk einlegen, um die Installation automatisch zu starten.

Sie können jetzt mit der Installation von Web Adaptor fortfahren.

Vorsicht:

Die Konfiguration von ArcGIS Web Adaptor für die Einbeziehung von zusätzlichen Level in die URL wird nicht unterstützt und kann zu Problemen bei Client-Anwendungen führen.

So installieren Sie ArcGIS Web Adaptor

Schritte:
  1. Das Installationsprogramm für ArcGIS Web Adaptor sollte automatisch starten, sobald der Download abgeschlossen ist. Wenn die Installation nicht automatisch startet, wechseln Sie zum Speicherort der heruntergeladenen Installationsdateien, und doppelklicken Sie auf die Datei Setup.exe.
  2. Überprüfen Sie die Bedingungen der Lizenzvereinbarung. Sie müssen den Bedingungen zustimmen, um fortfahren zu können.
  3. Wählen Sie für ArcGIS Web Adaptor eine Website aus, die auf Port 80 ausgeführt wird. Die verfügbaren Websites werden im Format <Websitename (Port)> aufgelistet. Wenn das Dialogfeld Website auswählen nicht angezeigt wird, bedeutet das, dass nur eine Website vorhanden ist. Wenn auf dem Computer nur eine Website gefunden wird, wird ArcGIS Web Adaptor automatisch auf der Website platziert, und das Dialogfeld Website auswählen wird nicht angezeigt.
  4. Geben Sie einen Namen für den Web Adaptor ein. Der Standardwert ist arcgis. Dieser Instanzname darf keine Leerzeichen enthalten.
    Hinweis:

    Wenn auf der ausgewählten Website bereits ein virtuelles Verzeichnis vorhanden ist, das den gleichen Namen wie der Web Adaptor hat, wird eine Meldung angezeigt. Klicken Sie auf OK, um die Meldung zu schließen, und geben Sie einen anderen Namen für Web Adaptor ein.

  5. Folgen Sie den Anweisungen am Bildschirm, um die Installation abzuschließen.

Die Konfigurationsseite für Web Adaptor wird angezeigt, wenn die Installation abgeschlossen ist. Sie können jetzt mit dem Konfigurieren von ArcGIS Web Adaptor für die Verwendung mit dem Portal beginnen.
