Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows Server 2008/2008 R2

Für ArcGIS Web Adaptor ist die Aktivierung von IIS und bestimmten IIS-Komponenten unter Windows Server 2008/2008 R2 erforderlich. Wenn IIS nicht gefunden wird und bestimmte IIS-Komponenten nicht aktiviert werden, wird die Installation nicht fortgesetzt.

Wenn IIS bereits installiert ist, die erforderlichen IIS-Komponenten jedoch fehlen, wird beim Installieren eine Meldung angezeigt, die darauf hinweist, dass bestimmte IIS-Komponenten fehlen. Sie haben die Möglichkeit, die erforderlichen Komponenten bei der Installation automatisch zu aktivieren. Je nach Sicherheitsrichtlinien Ihrer Organisation müssen die erforderlichen IIS-Komponenten jedoch möglicherweise manuell wie unten beschrieben aktiviert werden.

Informationen zur Aktivierung von IIS und den erforderlichen IIS-Komponenten unter Windows Server 2008/2008 R2 finden Sie in den folgenden Anweisungen.

  1. Klicken Sie auf Start > Verwaltung > Server Manager.

  2. Klicken Sie im linken Bereich des Dialogfeldes Server Manager auf Rollen.
    • Wenn IIS noch nicht aktiviert wurde, klicken Sie im Feld Rollenzusammenfassung auf Rollen hinzufügen. Klicken Sie auf Weiter, und aktivieren Sie Webserver (IIS) in der Liste. Klicken Sie anschließend erneut auf Weiter, und wählen Sie im linken Bereich die Option Rollendienste.
    • Wenn IIS bereits aktiviert ist, jedoch nicht alle erforderlichen Komponenten aktiviert wurden, klicken Sie im rechten Feld Webserver (IIS) auf Rollendienste hinzufügen.

  3. Überprüfen Sie im Dialogfeld Rollendienste auswählen, dass die unten aufgeführten Webserverkomponenten aktiviert sind.

  4. Nachdem die erforderlichen IIS-Komponenten aktiviert wurden, klicken Sie auf Weiter > Installieren.
