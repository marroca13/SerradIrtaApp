Erste Schritte mit ArcGIS Web Adaptor

Sie können ArcGIS Web Adaptor für die Verwendung mit ArcGIS Server oder mit Portal for ArcGIS konfigurieren. Die Konfiguration einer einzelnen ArcGIS Web Adaptor-Anwendung für die Verwendung mit beiden wird nicht unterstützt, obwohl dies technisch möglich ist. Wenn Sie sich für die Verwendung von ArcGIS Web Adaptor entschieden haben, finden Sie weitere Informationen zu den ersten Schritten in den nachfolgenden Abschnitten.

Verwenden von ArcGIS Web Adaptor mit ArcGIS Server

Wenn Sie den ArcGIS Web Adaptor nur mit ArcGIS Server verwenden möchten, finden Sie in den folgenden Themen Informationen zu den ersten Schritten:
