Automatisches Aktivieren von IIS-Komponenten

Die Installation von ArcGIS Web Adaptor erfordert bestimmte Komponenten der Microsoft Internetinformationsdienste (IIS) 7.0, 7.5, 8.0 oder 8.5. Wenn IIS 7.0, 7.5, 8.0 oder 8.5 bereits installiert ist, die erforderlichen IIS-Komponenten jedoch fehlen, wird beim Installieren eine Meldung angezeigt, die darauf hinweist, dass bestimmte IIS-Komponenten fehlen.

Sie haben die Möglichkeit, die erforderlichen IIS-Komponenten bei der Installation von Web Adaptor automatisch zu aktivieren. Alternativ können Sie die Komponenten selbst aktivieren. Wenn die erforderlichen IIS-Komponenten nicht aktiviert sind, wird die Installation nicht fortgesetzt.

Befolgen Sie die Anweisungen in den folgenden Abschnitten, um die fehlenden IIS-Komponenten automatisch oder manuell zu aktivieren.

Automatisches Aktivieren fehlender IIS-Komponenten

  1. Überprüfen Sie die fehlenden IIS-Komponenten im Dialogfeld Überprüfung der IIS-Anforderungen. Möglicherweise ist es für Sie hilfreich, die fehlenden Komponenten für Ihre Aufzeichnungen zu erfassen.
  2. Klicken Sie auf Ich stimme zu, um die fehlenden Komponenten automatisch zu aktivieren.

Die Komponenten werden aktiviert, und Sie können den Setupvorgang fortsetzen.

Manuelles Aktivieren von fehlenden IIS-Komponenten

Wenn Sie die fehlenden IIS 7.0-, 7.5-, 8.0- oder 8.5-Komponenten manuell aktivieren möchten, klicken Sie auf Abbrechen. Informationen zur manuellen Aktivierung fehlender IIS-Komponenten finden Sie in den unten aufgeführten Themen.

  • Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows Server 2012
  • Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows Server 2008/2008 R2
  • Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows 8/8.1
  • Aktivieren von IIS und den erforderlichen IIS-Komponenten unter Windows 7

Nachdem alle erforderlichen Komponenten aktiviert wurden, starten Sie die Installation von Web Adaptor neu, indem Sie im Setup-Ordner des Installationsprogramms für ArcGIS Web Adaptor (IIS) Setup.exe doppelklicken.
