Aktivieren von IIS und der erforderlichen IIS-Komponenten unter Windows Server 2012/2012 R2

Für ArcGIS Web Adaptor ist die Aktivierung von IIS und bestimmten IIS-Komponenten unter Windows Server 2012/2012 R2 erforderlich. Wenn IIS nicht gefunden wird und bestimmte IIS-Komponenten nicht aktiviert werden, wird die Installation nicht fortgesetzt.

Wenn IIS bereits installiert ist, die erforderlichen IIS-Komponenten jedoch fehlen, wird beim Installieren eine Meldung angezeigt, die darauf hinweist, dass bestimmte IIS-Komponenten fehlen. Sie haben die Möglichkeit, die erforderlichen Komponenten bei der Installation automatisch zu aktivieren. Je nach Sicherheitsrichtlinien Ihrer Organisation müssen die erforderlichen IIS-Komponenten jedoch möglicherweise manuell wie unten beschrieben aktiviert werden.

Informationen zur Aktivierung von IIS und der erforderlichen IIS-Komponenten unter Windows Server 2012/2012 R2 finden Sie in den folgenden Anweisungen.

  1. Öffnen Sie Server-Manager, und klicken Sie auf Verwalten > Rollen und Features hinzufügen. Klicken Sie auf Weiter.
  2. Wählen Sie Rollenbasierte oder featurebasierte Installation, und klicken Sie auf Weiter.
  3. Wählen Sie den geeigneten Server aus. Standardmäßig wird der lokale Server ausgewählt. Klicken Sie auf Weiter.
  4. Aktivieren Sie Web Server (IIS), und klicken Sie auf Weiter.
  5. Da zum Installieren des Web Adaptors keine zusätzlichen Funktionen erforderlich sind, klicken Sie auf Weiter.
  6. Klicken Sie im Dialogfeld Webserver-Rolle (IIS) auf Weiter.
  7. Überprüfen Sie im Dialogfeld Rollendienste auswählen, ob die unten aufgeführten Webserverkomponenten aktiviert sind. Klicken Sie auf Weiter.
  8. Überprüfen Sie, ob die Einstellungen korrekt sind, und klicken Sie auf Installieren.
  9. Klicken Sie nach Abschluss der Installation auf Schließen, um den Assistenten zu beenden.
