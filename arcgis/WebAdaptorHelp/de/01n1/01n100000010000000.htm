Aktivieren von IIS und der erforderlichen IIS-Komponenten unter Windows 8/8.1

Für ArcGIS Web Adaptor ist die Aktivierung von IIS und bestimmten IIS-Komponenten unter Windows 8/8.1 erforderlich. Wenn IIS nicht gefunden wird und bestimmte IIS-Komponenten nicht aktiviert werden, wird die Installation nicht fortgesetzt.

Wenn IIS bereits installiert ist, die erforderlichen IIS-Komponenten jedoch fehlen, wird beim Installieren eine Meldung angezeigt, die darauf hinweist, dass bestimmte IIS-Komponenten fehlen. Sie haben die Möglichkeit, die erforderlichen Komponenten bei der Installation automatisch zu aktivieren. Je nach Sicherheitsrichtlinien Ihrer Organisation müssen die erforderlichen IIS-Komponenten jedoch möglicherweise manuell wie unten beschrieben aktiviert werden.

Informationen zur Aktivierung von IIS und der erforderlichen IIS-Komponenten unter Windows 8/8.1 finden Sie in den folgenden Anweisungen.

  1. Öffnen Sie Systemsteuerung, und klicken Sie auf Programme und Funktionen > Windows-Funktionen aktivieren oder deaktivieren.
  2. Aktivieren Sie Internetinformationsdienste.
  3. Erweitern Sie die Funktion Internetinformationsdienste, und überprüfen, ob die unten aufgeführten Webserverkomponenten aktiviert sind.
  4. Klicken Sie auf OK.
