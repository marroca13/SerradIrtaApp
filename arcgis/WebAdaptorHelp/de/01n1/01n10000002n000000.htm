Konfigurieren der .NET-Vertrauensebene in IIS

ArcGIS Web Adaptor kann optional anhand der Microsoft .NET-Vertrauensebenen "Hoch" oder "Mittel" in Internetinformationssystemen (IIS) für die Kommunikation mit Portal for ArcGIS konfiguriert werden. Dies gilt für Organisationen mit höheren Sicherheitseinschränkungen, die eingeschränkte Berechtigungen für Anwendungen gewähren.

Sie müssen die Vertrauensebene auf "Voll" einstellen, wenn Sie Web Adaptor erstmals mit Portal for ArcGIS konfigurieren. Nachdem Sie die Konfiguration abgeschlossen und überprüft haben, ob Sie über die Web Adaptor-URL auf das Portal zugreifen können, haben Sie die Möglichkeit, den Web Adaptor so zu konfigurieren, dass er unter Verwendung der Vertrauensebenen "Hoch" oder "Mittel" mit dem Portal kommuniziert. Die Verwendung der Vertrauensebenen "Niedrig" und "Minimal" wird vom Web Adaptor nicht unterstützt. Die Vertrauensebene kann unabhängig davon eingestellt werden, ob der Web Adaptor-Anwendungspool auf Version 2.0 oder 4.0 in IIS festgelegt ist.

Info zu Microsoft .NET-Vertrauensebenen in IIS

In Internetinformationssystemen (IIS) werden die ihnen durch die Richtlinie .NET Codezugriffssicherheit (CAS, Code Access Security) gewährten Berechtigungen durch die Vertrauensebene einer Anwendung bestimmt. IIS ist standardmäßig mit der Vertrauensebene "Voll" konfiguriert, d. h. Anwendungen werden uneingeschränkte Berechtigungen gewährt. In einigen Organisationen mit höheren Sicherheitseinschränkungen werden Anwendungen nur als teilweise vertrauenswürdig eingestuft. Die teilweise Vertrauenswürdigkeit kann auf eine von vier Ebenen festgelegt werden: Hoch, Mittel, Niedrig und Minimal.

Info zum Verwenden von Microsoft .NET-Vertrauensebenen mit Web Adaptor

Wenn Sie den Web Adaptor installieren, wird die der Anwendung zugewiesene Vertrauensebene von der IIS-Website übernommen, für die Web Adaptor bereitgestellt wird. Dieses Verhalten ist identisch, unabhängig davon, ob der Web Adaptor-Anwendungspool auf Version 2.0 oder 4.0 in IIS festgelegt ist.

Wenn der Website die Vertrauensebene "Voll" zugewiesen ist, sind keine weiteren Maßnahmen erforderlich. Sie können den Web Adaptor normal mit Portal for ArcGIS konfigurieren. Nach Abschluss der Konfiguration können Sie die Vertrauensebene der Anwendung wieder auf die ursprüngliche Ebene zurücksetzen und wie gewohnt über den Web Adaptor auf Ihr Portal zugreifen.

Die Verwendung der Vertrauensebenen "Niedrig" und "Minimal" wird von der ArcGIS Web Adaptor-Anwendung nicht unterstützt. Sie können weder auf die Konfigurationsseite zugreifen noch mit Portal for ArcGIS kommunizieren. Da die Kommunikation mit der ArcGIS-Software auf den Ebenen "Niedrig" und "Mittel" möglich ist, wird lediglich eine IIS-Fehlerseite angezeigt.

Nachdem Web Adaptor konfiguriert wurde, kann die Vertrauensebene auf "Hoch" oder "Mittel" eingestellt werden. Die Verwendung des Web Adaptors zum Weiterleiten von Anforderungen wird auf den Vertrauensebenen "Hoch" oder "Mittel" unterstützt. Führen Sie die folgenden Schritte aus, um die Vertrauensebene zu ändern.

Ändern der Vertrauensebene des Web Adaptors

Schritte:
  1. Öffnen Sie Internetinformationsdienste (IIS)-Manager.
  2. Erweitern Sie im Fenster Verbindungen die Option Sites.
  3. Erweitern Sie die Website, auf der der Web Adaptor bereitgestellt wird, und wählen Sie die Web Adaptor-Anwendung aus.
  4. Doppelklicken Sie auf .NET-Vertrauensebenen.
  5. Wählen Sie aus der Dropdown-Liste Vertrauensebene ggf. Voll (intern), Hoch oder Mittel aus.
  6. Klicken Sie auf Übernehmen.
