Aufheben der Registrierung von ArcGIS Web Adaptor bei Portal for ArcGIS

Wenn Sie Portal for ArcGIS für die Verwendung eines anderen ArcGIS Web Adaptors ändern möchten, d. h. beispielsweise den Namen des Web Adaptors oder den Namen des Computers, der ArcGIS Web Adaptor hostet, ändern, müssen Sie zuerst die Registrierung von ArcGIS Web Adaptor bei Ihrem Portal aufheben.

Verwenden Sie ArcGIS Portal Directory, um die Registrierung von ArcGIS Web Adaptor bei Ihrem Portal aufzuheben. Die Aufhebung der Registrierung kann nur über ArcGIS Portal Directory erfolgen; die Funktion zum Aufheben der Registrierung ist auf der Konfigurationsseite von ArcGIS Web Adaptor bzw. auf der Portal-Website nicht verfügbar.

Führen Sie zum Aufheben der Registrierung von ArcGIS Web Adaptor die folgenden Schritte aus.

Schritte:
  1. Öffnen Sie das ArcGIS Portal Directory, und melden Sie sich als Administrator Ihrer Organisation an. Die URL hat das Format https://portal.domain.com:7443/arcgis/portaladmin.
  2. Klicken Sie auf System > Web Adaptor.
  3. Klicken Sie auf der Seite Web Adaptor auf den Namen des Computers, der ArcGIS Web Adaptor hostet.
  4. Klicken Sie auf der Seite Web Adaptor: <Computername> auf Registrierung von Web Adaptor aufheben.
  5. Klicken Sie auf der Seite Registrierung von Web Adaptor aufheben auf Registrierung von Web Adaptor aufheben. Die Registrierung von ArcGIS Web Adaptor bei Ihrem Portal wird aufgehoben.
  6. Konfigurieren Sie den neuen ArcGIS Web Adaptor in Ihrem Portal. Weitere Informationen finden Sie unter Konfigurieren von ArcGIS Web Adaptor.
Hinweis:

Wenn Sie die vorherige Version von ArcGIS Web Adaptor vollständig aus dem Portal entfernen möchten, wird empfohlen, ArcGIS Web Adaptor zu deinstallieren, nachdem Sie die Schritte oben ausgeführt haben. Beim Deinstallieren werden sämtliche Konfigurationsdetails von ArcGIS Web Adaptor aus dem Portal gelöscht und der Zugriff auf das Portal über die nicht registrierte ArcGIS Web Adaptor-URL wird für Mitglieder und Clients gesperrt. Weitere Informationen finden Sie unter Deinstallieren von ArcGIS Web Adaptor.
