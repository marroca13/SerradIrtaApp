安装多个 ArcGIS Web Adaptor

在同一台计算机上可同时安装多个 ArcGIS Web Adaptor。也可以在不同的计算机上重复此安装。以下步骤将指导您完成整个安装过程。

如何安装多个 Web Adaptor

可以同时在同一台计算机上安装多个 Web Adaptor，或在多个单独的计算机上执行此安装。如果要在同一个站点（端口）安装多个 Web Adaptor，它们需要使用不同的名称。例如，名为 arcgis 的两个 Web Adaptor 不能存在于同一个网站中。如果需要多个同名 Web Adaptor，则需在不同的站点（端口）安装它们。

  1. 浏览到包含 ArcGIS Web Adaptor (IIS) 安装程序的下载文件夹，或者将 ArcGIS Server 安装介质插入相应的驱动器，以自动启动介质运行界面。
  2. 按照安装第一个 Web Adaptor 时的说明进行操作。有关详细信息，请参阅安装 ArcGIS Web Adaptor。
注：

安装程序将检测并防止 Web Adaptor 名称冲突。如果显示一条消息，提示虚拟目录已存在，请为 Web Adaptor 指定不同的名称，或选择一个不同的站点。

注：

要应用 Web Adaptor 补丁包，则不能移除基础实例。

如何静默安装多个 Web Adaptor

可通过使用 Windows Installer 命令行参数运行安装程序来安装多个 Web Adaptor。有关如何使用命令行参数的详细说明，请参阅静默安装 ArcGIS Web Adaptor。安装多个 Web Adaptor 无需其他命令行参数。

如何卸载多个 Web Adaptor

必须分别卸载每个 Web Adaptor。有关完整说明，请参阅卸载 ArcGIS Web Adaptor。
