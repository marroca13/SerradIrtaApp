自动启用 IIS 组件

要安装 ArcGIS Web Adaptor，需使用某些 Microsoft Internet 信息服务 (IIS) 7.0、7.5、8.0 或 8.5 组件。如果您已启用 IIS 7.0、7.5、8.0 或 8.5，但缺少所需 IIS 组件，则安装程序将显示一条消息，提示缺少某些 IIS 组件。

您可以选择允许 Web Adaptor 安装程序自动启用所需 IIS 组件。也可以自行启用这些组件。如果未启用所需 IIS 组件，则安装将无法继续。

请按照以下说明自动或手动启用缺少的 IIS 组件。

自动启用缺少的 IIS 组件

  1. 在 IIS 要求验证 对话框中查看缺少的 IIS 组件。您可能发现这样有助于记录缺少的组件，
  2. 要自动启用缺少的组件，请单击我同意。

将启用组件并继续安装过程。

手动启用缺少的 IIS 组件

如果要手动启用缺少的 IIS 7.0、7.5、8.0 或 8.5 组件，请单击取消。要了解如何自行手动启用缺少的 IIS 组件，请参阅以下主题。

  • 在 Windows Server 2012 中启用 IIS 及所需 IIS 组件
  • 在 Windows Server 2008/2008 R2 中启用 IIS 及所需 IIS 组件
  • 在 Windows 8/8.1 中启用 IIS 及所需 IIS 组件
  • 在 Windows 7 中启用 IIS 及所需 IIS 组件

启用所有所需组件后，双击 ArcGIS Web Adaptor (IIS) 安装程序文件夹中的 Setup.exe 再次启动 Web Adaptor 安装程序。
