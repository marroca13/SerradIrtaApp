配置多个 ArcGIS Web Adaptor

配置使用 Web 层身份验证的高可用性门户时，会需要多个 ArcGIS Web Adaptor。要使用 Portal for ArcGIS 配置多个 Web Adaptor，需要使用单一端点来访问门户。必须在 WebContextURL 属性中定义该 URL。然后即可按照下列步骤使用门户对多个 Web Adaptor 进行配置。

步骤：
  1. 通过门户配置第一个 ArcGIS Web Adaptor。
  2. 将 Portal for ArcGIS 添加到反向代理服务器。 确保已按下列步骤设置 WebContextURL 属性：
    1. 打开 Web 浏览器并以组织管理员的身份登录到 ArcGIS Portal Directory。URL 地址格式为 https://portal.domain.com:7443/arcgis/portaladmin。
    2. 单击系统 > 属性 > 更新属性。这将导致 Portal for ArcGIS 服务重启。您将需要等待重新启动完成后才能继续操作。
    3. 在更新系统属性对话框中，插入以下 JSON，将您自己的反向代理服务器或 DNS 别名 URL 替换为组织防火墙以外的用户所看到的 URL。
      {
   "WebContextURL": "https://reverseproxy.domain.com/myorg"
}
    4. 单击更新属性。
  3. 配置附加 Web Adaptor。

通过门户配置的所有 Web Adaptor 都将在配置摘要页面中列出。用户可通过反向代理服务器或 DNS 别名 URL 访问门户网站。
