安装 ArcGIS Web Adaptor

ArcGIS Web Adaptor 以安装程序的形式提供给客户，并通过名为 Setup.exe 的实用程序执行安装。可按照此处提供的步骤独立安装 ArcGIS Web Adaptor，或使用 Windows Installer 命令行参数静默安装。

安装行为和要求

ArcGIS Web Adaptor 版本必须始终与其注册的 ArcGIS Server 站点版本相匹配。

ArcGIS Web Adaptor 可以与其他版本并行安装。无需卸载现有 ArcGIS Web Adaptor，即可安装新 ArcGIS Web Adaptor，且 ArcGIS Web Adaptor 并非之前版本的升级。

各个 ArcGIS Web Adaptor 的名称必须唯一，单个 web 服务器上不能有两个名称相同的 Web Adaptor。若在安装过程中检测到命名冲突，则会显示警告消息。若要解决此冲突，需要卸载同名的较早版本。或者，可以更改其中一个 Web Adaptor 的名称，然后继续安装。

升级早期版本的 ArcGIS Web Adaptor

要将早期版本的 ArcGIS Web Adaptor 升级为 10.6，您需要卸载早期版本，然后运行 10.6 安装程序。在安装 10.6 版本期间，需指定与较早版本 Web Adaptor 相同的名称。这可保证您的 Web Adaptor URL 将与较早版本中的 URL 保持一致。

若在 10.6 安装过程中检测到命名冲突，则会显示警告消息。若要执行升级，需要卸载同名的较早版本。

安装 10.6 后，需要使用相应的 ArcGIS Web Adaptor 站点配置 ArcGIS Server。

旧版本：

对于 10.2 和较早版本，运行安装程序会将所有现有的 Web Adaptor 升级为最新版本。

安装 ArcGIS Web Adaptor 前

  1. 验证系统是否满足所有系统要求。
  2. 以具有管理权限的用户身份登录。
  3. 关闭所有应用程序。
  4. 浏览至包含 ArcGIS Web Adaptor 安装程序的下载文件夹。也可以将 ArcGIS Server 介质插入到相应的驱动器以自动启动安装程序。

接下来即可开始执行 Web Adaptor 的安装。

注：

尝试配置 ArcGIS Web Adaptor 会使包括其他等级的 URL 不受支持，并可能导致客户端应用程序出现问题。

如何安装 ArcGIS Web Adaptor

  1. 下载完成后，ArcGIS Web Adaptor 安装程序将自动启动。如果安装程序未自动启动，请浏览至已下载的安装程序文件的位置，并双击 Setup.exe。

  2. 阅读主协议的条款和条件。必须同意条款才能继续安装。

  3. 为 Web Adaptor 选择网站。可用网站将以<网站名称（端口）>的形式列出。如果未显示选择站点 对话框，则表示当前只有一个站点可供使用。如果您的计算机上只存在一个网站，则 ArcGIS Web Adaptor 将被自动放置在该站点中，而不会显示选择网站 对话框。

  4. 输入 Web Adaptor 的名称。默认值为 arcgis。此实例名称不能包含空格。

    注：

    如果所选网站中已经存在与 Web Adaptor 名称相同的虚拟目录，则将显示一条消息。单击确定关闭此消息，然后为 Web Adaptor 提供一个不同的名称。

  5. 要完成安装，请按照屏幕上的说明执行后续操作。

安装完成后，随即显示 Web Adaptor 配置页面。接下来即可配置 ArcGIS Web Adaptor 以与 ArcGIS Server 配合使用。
