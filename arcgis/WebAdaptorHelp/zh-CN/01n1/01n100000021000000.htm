使用 Portal for ArcGIS 执行 ArcGIS Web Adaptor 的概述

使用 Portal for ArcGIS 执行 ArcGIS Web Adaptor 需要遵循的步骤概述如下:

步骤：
  1. 规划如何在 Portal for ArcGIS 中使用 ArcGIS Web Adaptor。
  2. 验证系统是否满足最低系统要求。
  3. 在 Web 服务器上启用 HTTPS
  4. 安装 ArcGIS Web Adaptor。
  5. 配置 ArcGIS Web Adaptor 以与 Portal for ArcGIS 一起使用。
