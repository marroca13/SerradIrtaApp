使用 ArcGIS Server 注销 ArcGIS Web Adaptor

如果您想要修改 ArcGIS Server，以使用其他的 ArcGIS Web Adaptor，例如您想要更改 Web Adaptor 的名称或托管 ArcGIS Web Adaptor 的计算机名称，您首选需要使用 ArcGIS Serve 站点注销 ArcGIS Web Adaptor。

您将使用ArcGIS Server 管理员目录来使用 ArcGIS Server 站点注销 ArcGIS Web Adaptor。您只能通过 ArcGIS Server 管理员目录注销；可从 ArcGIS Web Adaptor 配置页面处访问注销功能。

要注销 ArcGIS Web Adaptor，请执行以下步骤。

步骤：
  1. 打开 ArcGIS Server 管理员目录，并以站点管理员的身份进行登录。URL 地址格式为 https://gisserver.domain.com:6443/arcgis/admin。
  2. 单击系统 > Web Adaptor。
  3. 在 Web Adaptors 页面，单击代表 ArcGIS Web Adaptor 的字符串。
  4. 在 Web Adaptor 页面，确保 URL 列出的内容涉及的是您想要注销的 Web Adaptor。单击取消注册。
  5. 在注销 Web Adaptor 页面上，单击注销 Web Adaptor。ArcGIS Web Adaptor 即从您的 ArcGIS Server 站点注销。
  6. 使用 ArcGIS Server 配置新的 ArcGIS Web Adaptor。有关详细说明，请参阅配置 ArcGIS Web Adaptor。
注：

如果您希望从 ArcGIS Server 站点完全移除先前版本的 ArcGIS Web Adaptor，则建议您在执行上述步骤后卸载 ArcGIS Web Adaptor。卸载将从站点中删除所有的 ArcGIS Web Adaptor 配置详细信息，并阻止成员和客户端通过注销的 ArcGIS Web Adaptor URL 访问站点。有关详细说明，请参阅卸载 ArcGIS Web Adaptor。
