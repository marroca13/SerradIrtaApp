下面列出了运行 ArcGIS Web Adaptor 所需的系统规范和硬件要求。有关其他要求和更新要求，请参阅 Web 上提供的系统要求。 有关先前版本的信息，请参阅 Esri 支持。

建议您检查弃用通知，以确定硬件和软件组件是否仍与 10.6 版本兼容。

最多安装实例

最多可在单个计算机上安装 21 个 ArcGIS Web Adaptor 实例。如果需要更多实例，您需要在另一台计算机上进行安装。

如果您的计算机上安装了较早版本，无须卸载它们。例如，在同一台计算机上可以安装 5 个 ArcGIS Web Adaptor 10.4 实例和 21 个 ArcGIS Web Adaptor 10.6 实例。最大值仅适用于相同软件版本的实例。

Microsoft .NET Framework 4.5 支持

需要 Microsoft .NET Framework 4.5 或更高版本才能安装 ArcGIS Web Adaptor。如果您的操作系统已安装 Microsoft .NET Framework 4.5，则请确保已使用 Windows 功能将其启用。如果您的操作系统未安装 Microsoft .NET Framework 4.5，则可从安装介质上进行下载并使用 Windows 功能将其启用。

Microsoft IIS 所需的组件

ArcGIS Web Adaptor 支持 Windows 7、8.1、10、Server 2008 R2、Server 2012 和 2012 R2 中的 IIS。这与 7.0、7.5、8.0、8.5 和 10 版本的 IIS 相对应。IIS 必须与特定 IIS 7.0、7.5、8.0、8.5 和 10 组件一同启用。如果未检测到 IIS 或未启用特定的 IIS 组件，则安装程序将无法继续。

如果已安装 IIS 7.0、7.5、8.0 或 8.5，但是缺少所需的 IIS 组件，则安装程序将显示 IIS 要求验证 对话框。您可以在其中选择允许安装程序自动启用所有缺少的必需 IIS 组件。为此，请单击我同意。要了解详细信息，请参阅自动启用 IIS 组件。

注：

如果您打算静默安装 ArcGIS Web Adaptor，则必须手动启用所有所需 IIS 组件。在执行静默安装时，安装程序不会自动启用缺少的 IIS 组件。

要了解如何在您的操作系统中启用 IIS 以及如何有选择地自行启用缺少的 IIS 7.0、7.5、8.0 或 8.5 组件，请参阅以下主题：

  • 在 Windows Server 2012/2012 R2 中启用 IIS 及所需 IIS 组件
  • 在 Windows Server 2008 R2 中启用 IIS 及所需 IIS 组件
  • 在 Windows 8.1 中启用 IIS 及所需 IIS 组件
  • 在 Windows 7 中启用 IIS 及所需 IIS 组件
