在 Portal for ArcGIS 上注销 ArcGIS Web Adaptor

如果您想要修改 Portal for ArcGIS 以使用其他的 ArcGIS Web Adaptor，例如，想要更改 Web Adaptor 的名称或托管 ArcGIS Web Adaptor 的计算机名称，则首先需要在门户中注销 ArcGIS Web Adaptor。

您可在您的门户中使用 ArcGIS Portal 目录注销 ArcGIS Web Adaptor。只能通过 ArcGIS 门户目录注销，注销功能在 ArcGIS Web Adaptor 配置页面或门户网站中未提供。

要注销 ArcGIS Web Adaptor，请执行以下步骤。

步骤：
  1. 打开 ArcGIS 门户目录，并以组织管理员的身份进行登录。URL 地址格式为 https://portal.domain.com:7443/arcgis/portaladmin。
  2. 单击系统 > Web Adaptor。
  3. 在 Web Adaptors 页面上，单击托管 ArcGIS Web Adaptor 的计算机名称。
  4. 在 Web Adaptor：<计算机名称> 页面上，单击注销 Web Adaptor。
  5. 在注销 Web Adaptor 页面上，单击注销 Web Adaptor。已在您的门户中注销 ArcGIS Web Adaptor。
  6. 在门户中配置新的 ArcGIS Web Adaptor。有关详细说明，请参阅配置 ArcGIS Web Adaptor。
注：

如果您希望从门户上将先前版本的 ArcGIS Web Adaptor 完全移除，则建议您执行上述步骤后，卸载 ArcGIS Web Adaptor。卸载将从门户中删除 ArcGIS Web Adaptor 的所有配置详细信息并阻止成员和客户端通过已注销的 ArcGIS Web Adaptor URL 访问门户。有关详细说明，请参阅卸载 ArcGIS Web Adaptor。
