配置 ArcGIS Web Adaptor 的显示语言

ArcGIS Web Adaptor 安装程序包括多种语言的 Web Adaptor 预本地化版本。预本地化的语言如下：

  • 阿拉伯语
  • 简体中文
  • 法语
  • 德语
  • 意大利语
  • 日语
  • 巴西葡萄牙语
  • 俄语
  • 西班牙语
  • 土耳其语

安装程序完成后会自动打开 Web Adaptor 配置页面。如果浏览器已配置为显示上述任一语言，则 Web Adaptor 应用程序将自动以该语言打开。

要以不同的语言显示 Web 适配器，请按以下步骤对浏览器或系统进行配置。

Firefox 用户

支持本地化的 Firefox 版本。安装 Firefox 的本地化版本并打开 Web Adaptor 后，Web Adaptor 将自动以浏览器语言显示。

如果想使用 Firefox 并以不同于浏览器语言的其他语言显示 Web Adaptor，则必须使用 Firefox 10 或更高版本，并进行以下配置：

  1. 选择工具 > 选项。
  2. 在内容选项卡中，单击语言下的选择。
  3. 选择想要显示的语言，并将其移动到语言列表的顶部。

Chrome 用户

如果您使用的是 Google Chrome，若要修改 Web Adaptor 应用程序的显示语言，请进行以下配置：

  1. 选择自定义 > 选项 > 高级选项。
  2. 在“Web 内容”部分中，单击语言和拼写检查选项。
  3. 选择要显示的语言，然后单击以该语言显示 Google Chrome。

Internet Explorer 用户

如果您使用的是 Internet Explorer，若要修改 Web Adaptor 的显示语言，请进行以下配置：

  1. 选择工具 > Internet 选项。
  2. 在常规选项卡中，单击外观下的语言按钮。
  3. 选择想要显示的语言，并将其移动到语言列表的顶部。单击确定，然后再单击确定以接受更改。
