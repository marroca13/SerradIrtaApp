Configurar el nivel de confianza .NET en IIS

ArcGIS Web Adaptor se puede modificar para que se comunique con Portal for ArcGIS utilizando el nivel de confianza Alto o Medio de Microsoft .NET en Internet Information Systems (IIS). Esto se aplica a organizaciones que cuentan con más restricciones de seguridad para conceder permisos limitados a las aplicaciones.

Debe establecer el nivel de confianza en Completo cuando configure por primera vez el Web Adaptor con Portal for ArcGIS. Cuando haya finalizado la configuración y haya comprobado que puede acceder al portal mediante la URL de Web Adaptor, puede modificar el Web Adaptor para comunicar con el portal utilizando el nivel de confianza Alto o Medio a partir de ese momento. Web Adaptor no admite el uso de los niveles de confianza Bajo y Mínimo. El nivel de confianza se puede establecer independientemente de que el grupo de aplicaciones de Web Adaptor esté establecido con la versión 2.0 o 4.0 en IIS.

Acerca de los niveles de confianza de Microsoft .NET en IIS

En Internet Information Systems (IIS), el nivel de confianza de una aplicación determina los permisos que tiene concedidos mediante la directiva de seguridad de acceso al código (CAS) de .NET. De forma predeterminada, IIS está configurada con el nivel de confianza Completo, lo que significa que no hay restricciones de permisos para las aplicaciones. Sin embargo, en algunas organizaciones que cuentan con más restricciones de seguridad, solo se concede un nivel de confianza parcial a las aplicaciones. Al nivel de confianza Parcial se le puede asignar uno de estos cuatro niveles: Alto, Medio, Bajo y Mínimo.

Acerca de cómo se utilizan los niveles de confianza de Microsoft .NET con Web Adaptor

Al instalar el Web Adaptor, el nivel de confianza asignado a la aplicación se hereda del sitio web de IIS en el que se implementa el Web Adaptor. Este comportamiento es idéntico independientemente de que el grupo de aplicaciones de Web Adaptor esté establecido con la versión 2.0 o 4.0 en IIS.

Si el sitio web tiene asignado un nivel de confianza Completo, no necesita hacer nada más. Puede configurar Web Adaptor con Portal for ArcGIS de la forma habitual. Antes de configurar ArcGIS Web Adaptor con Portal for ArcGIS debe establecer el nivel de confianza de .NET para la aplicación del Web Adaptor con el valor Completo en IIS. Después de configurarlo, puede restablecer el nivel de confianza original de la aplicación y acceder al portal mediante Web Adaptor de la forma habitual.

La aplicación ArcGIS Web Adaptor no admite el uso de los niveles de confianza Bajo y Mínimo. No podrá acceder a la página de configuración ni comunicarse con Portal for ArcGIS. Dado que no es posible comunicarse con el software de ArcGIS utilizando los niveles Bajo y Mínimo, solo verá una página de error de IIS.

Debe establecer el nivel de confianza en Completo cuando configure por primera vez el Web Adaptor con Portal for ArcGIS. Una vez configurado, puede modificar el Web Adaptor con el nivel de confianza Alto o Medio. Con el nivel de confianza Alto o Medio se puede utilizar el Web Adaptor para reenviar solicitudes. Siga los pasos siguientes para modificar el nivel de confianza.

Modificar el nivel de confianza del Web Adaptor

Pasos:
  1. Abra el Administrador de Internet Information Systems (IIS).
  2. En el panel Conexiones, expanda la opción Sitios.
  3. Expanda el sitio web en el que está implementado Web Adaptor y seleccione la aplicación Web Adaptor.
  4. Haga doble clic en Niveles de confianza de .NET.
  5. En la lista desplegable de Nivel de confianza, seleccione Completo (interno), Alto o Medio, según sea necesario.
  6. Haga clic en Aplicar.
