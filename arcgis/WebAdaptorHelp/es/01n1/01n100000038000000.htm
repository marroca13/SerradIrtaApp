Dar de baja ArcGIS Web Adaptor con ArcGIS Server

Si desea modificar ArcGIS Server para utilizar un ArcGIS Web Adaptor diferente, por ejemplo, si desea cambiar el nombre del Web Adaptor o el nombre del equipo que aloja ArcGIS Web Adaptor, primero tendrá que dar de baja ArcGIS Web Adaptor con su sitio de ArcGIS Server.

Debe utilizar el Directorio de administrador de ArcGIS Server para dar de baja ArcGIS Web Adaptor con su sitio de ArcGIS Server. Solo puede darlo de baja mediante el Directorio de administrador de ArcGIS Server; la funcionalidad de dar de baja no está disponible desde la página de configuración de ArcGIS Web Adaptor.

Siga estos pasos para dar de baja ArcGIS Web Adaptor.

Pasos:
  1. Abra el Directorio de administrador de ArcGIS Server e inicie sesión como administrador del sitio. La dirección URL tiene el formato https://gisserver.domain.com:6443/arcgis/admin.
  2. Haga clic en Sistema > Web Adaptors.
  3. En la página Web Adaptors, haga clic en la cadena de caracteres que representa ArcGIS Web Adaptor.
  4. En la página Web Adaptor, asegúrese de que la dirección URL que se muestra hace referencia al Web Adaptor que desea dar de baja. Haga clic en Dar de baja.
  5. En la página Dar de baja Web Adaptor, haga clic en Dar de baja Web Adaptor. ArcGIS Web Adaptor se ha dado de baja de su sitio de ArcGIS Server.
  6. Configure el nuevo ArcGIS Web Adaptor con su sitio de ArcGIS Server. Para ver las instrucciones completas, consulte Configurar ArcGIS Web Adaptor.
Nota:

Si su intención es eliminar por completo el anterior ArcGIS Web Adaptor de su sitio de ArcGIS Server, se recomienda desinstalar ArcGIS Web Adaptor después de seguir los pasos anteriores. La desinstalación elimina todos los detalles de configuración de ArcGIS Web Adaptor del sitio y bloquea el acceso al sitio de los miembros y clientes mediante la dirección URL de ArcGIS Web Adaptor que se ha dado de baja. Para ver las instrucciones completas, consulte Desinstalar ArcGIS Web Adaptor.
