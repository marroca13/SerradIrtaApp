Habilitar IIS y los componentes obligatorios de IIS en Windows 7

ArcGIS Web Adaptor requiere que IIS esté habilitado y que determinados componentes de IIS también lo estén en Windows 7. No se podrá proceder con la instalación si no se detecta IIS y los componentes de IIS específicos no están habilitados.

Si ya ha habilitado IIS pero faltan los componentes necesarios de IIS, el proceso de instalación mostrará un mensaje indicando que faltan ciertos componentes de IIS. Tiene la opción de permitir que la instalación habilite automáticamente los componentes de IIS obligatorios. Sin embargo, dependiendo de las políticas de seguridad de su organización, tal vez sea necesario habilitar manualmente los componentes requeridos de IIS, tal y como se describe a continuación.

Para aprender a habilitar IIS y los componentes de IIS necesarios en Windows 7, consulte las instrucciones siguientes.

  1. Abra Panel de control > Programas y Características > Activar o desactivar las características de Windows.

  2. Expanda la característica Internet Information Services.

  3. Verifique que los componentes de IIS que se enumeran a continuación estén activados. Si estos componentes no están habilitados, no se realizará la instalación.

Componentes de IIS necesarios

Los componentes de IIS que se detallan a continuación cumplen con los requisitos mínimos para ejecutar Web Adaptor. Si están habilitados otros componentes de IIS, no es necesario quitarlos.
