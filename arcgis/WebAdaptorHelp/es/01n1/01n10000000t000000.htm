Información general sobre cómo importar ArcGIS Web Adaptor con ArcGIS Server

A continuación se describen de forma general los pasos que es preciso seguir para implementar Web Adaptor con ArcGIS Server:

  1. Planifique la configuración de su sitio de ArcGIS Server.
  2. Verifique que el sistema cumpla con los requisitos mínimos del sistema.
  3. Instale ArcGIS Web Adaptor.
  4. Configurar ArcGIS Web Adaptor para usar con ArcGIS Server.
