Introducción a ArcGIS Web Adaptor

Puede configurar ArcGIS Web Adaptor para que funcione con ArcGIS Server o con Portal for ArcGIS. No se permite configurar una sola aplicación de ArcGIS Web Adaptor para que funcione con ambos, aunque es técnicamente posible. Una vez que decida cómo va a usar ArcGIS Web Adaptor, consulte las secciones siguientes para dar los primeros pasos.

  • Vista general de la implementación de Web Adaptor con ArcGIS Server
