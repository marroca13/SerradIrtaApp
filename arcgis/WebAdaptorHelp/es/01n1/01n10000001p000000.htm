Configurar ArcGIS Web Adaptor

Después de instalar ArcGIS Web Adaptor, debe configurarlo para trabajar con su portal de ArcGIS Enterprise. Esto se puede hacer desde una página web o mediante la utilidad línea de comandos instalada con ArcGIS Web Adaptor. Como medida de seguridad, solo se puede acceder a la página de configuración desde el equipo donde está alojado ArcGIS Web Adaptor.

Al abrir la página de configuración se muestra el estado actual de ArcGIS Web Adaptor. Esto indica si ArcGIS Web Adaptor se configuró previamente. Para ver el tipo de plataforma y el número de versión de ArcGIS Web Adaptor, haga clic en Acerca en la página de configuración.

Si ya ha configurado un Web Adaptor con su portal de ArcGIS Enterprise y desea actualizar el portal para utilizar otro Web Adaptor, primero deberá dar de baja el Web Adaptor actual. Esto solo se puede hacer desde el Directorio de Portal. La funcionalidad de dar de baja no está disponible desde la página de configuración de Web Adaptor ni desde el sitio web del portal. Para ver las instrucciones completas, consulte Dar de baja ArcGIS Web Adaptor con Portal for ArcGIS.

Para obtener instrucciones completas sobre cómo configurar ArcGIS Web Adaptor, siga los pasos que figuran más abajo.

Precaución:

Antes de continuar, asegúrese de que ArcGIS Web Adaptor esté instalado en un sitio web que se ejecute en el puerto 80 o 443. ArcGIS Web Adaptor no se puede configurar con Portal for ArcGIS si está instalado en un sitio web que funcione en un puerto distinto del 80 o el 443.

Configurar ArcGIS Web Adaptor desde la página web de configuración

Para configurar ArcGIS Web Adaptor desde la página web de configuración instalada con el software, siga los pasos a continuación.

Pasos:
  1. Si no lo ha hecho todavía, habilite HTTPS en el puerto 443 del servidor web. Para obtener más información, consulte Activar HTTPS en el servidor web.
  2. Abra la página de configuración en un navegador web utilizando una dirección URL de HTTPS, como https://webadaptorhost.domain.com/webadaptorname/webadaptor.

    Si se va a usar un alias de DNS con el portal, el Web Adaptor debe configurarse mediante el alias de DNS en su lugar, usando una URL como https://<dnsalias.domain.com>/<webadaptorname>/webadaptor.

  3. Elija Portal for ArcGIS y haga clic en Siguiente.
  4. Para URL del portal, escriba la dirección URL en el equipo donde está alojado Portal for ArcGIS. Incluya el nombre de dominio completo de la máquina en la dirección URL, por ejemplo, https://portal.domain.com:7443.
  5. Especifique el nombre de usuario y la contraseña de una cuenta que tenga privilegios administrativos en Portal for ArcGIS. Normalmente se utilizan el nombre de usuario y la contraseña de la cuenta del administrador inicial definida al crear el portal por primera vez. Si degradó o eliminó la cuenta del administrador inicial, deberá especificar una cuenta con acceso de administración al sitio web del portal.
  6. Haga clic en Configurar.

ArcGIS Web Adaptor está configurado para usarse con el equipo donde se aloja Portal for ArcGIS. Ahora, podrá acceder a su portal a través de la dirección URL de ArcGIS Web Adaptor, en lugar del puerto 7443. La dirección URL tiene el formato siguiente: https://webadaptorhost.domain.com/webadaptorname/home.

Si ha configurado el portal para que utilice HTTPS para todas las comunicaciones, debería actualizar el sitio web del portal instalado y procurar que las direcciones URL de los accesos directos usen https en lugar de http. De lo contrario, se producirán fallos en el navegador al tratar de acceder a las direcciones URL de accesos directos originales.

Si va a configurar Portal for ArcGIS con el servidor proxy inverso de su organización, tendrá que proporcionar alguna información al portal acerca del servidor proxy. Para obtener instrucciones completas, consulte Configurar un servidor proxy inverso con el portal.

Configurar ArcGIS Web Adaptor desde la línea de comandos

Puede configurar ArcGIS Web Adaptor (IIS) mediante la utilidad de la línea de comandos (ConfigureWebAdaptor.exe) de C:\Program Files (x86)\Common Files\ArcGIS\WebAdaptor\IIS\Tools.

Los parámetros disponibles son los siguientes:

ConfigureWebAdaptor.exe /m <Mode> /w <WebAdaptorURL> /g <URL> /u <Username> /p <Password>

  • <Modo>: modo de producto en el que se configurará ArcGIS Web Adaptor. De forma predeterminada, este parámetro se establece en server, lo que significa que ArcGIS Web Adaptor se configurará para usarse con ArcGIS Server. Dado que está configurando ArcGIS Web Adaptor para usarse con Portal for ArcGIS, especifique este parámetro como portal.

  • <WebAdaptorURL>: dirección URL del ArcGIS Web Adaptor que se desea configurar desde la línea de comandos. Es la misma dirección URL que se utilizaría para acceder a la página de configuración de Web Adaptor desde un navegador web. Es necesario especificar la dirección URL HTTPS del ArcGIS Web Adaptor, por ejemplo, https://webadaptorhost.domain.com/webadaptorname/webadaptor. Si se va a usar un alias de DNS con el portal, el Web Adaptor debe configurarse mediante el alias de DNS en su lugar, usando una URL como https://<dnsalias.domain.com>/<webadaptorname>/webadaptor.

  • <URL>: dirección URL del equipo donde se aloja Portal for ArcGIS. Incluya el nombre de dominio completo de la máquina en la dirección URL, por ejemplo, https://portal.domain.com:7443.

  • <Username>: nombre de usuario de una cuenta con privilegios de administración sobre Portal for ArcGIS. Normalmente se utiliza la cuenta del administrador inicial definida al crear el portal por primera vez.

  • <Password>: contraseña de una cuenta con privilegios de administración sobre Portal for ArcGIS. Normalmente se utiliza la cuenta del administrador inicial definida al crear el portal por primera vez.

Ejemplo: ConfigureWebAdaptor.exe /m portal /w https://webadaptorhost.domain.com/webadaptorname/webadaptor /g https://portal.domain.com:7443 /u initialadmin /p secret

Una vez ejecutado el comando, ArcGIS Web Adaptor está ya configurado para usarse con el equipo donde se aloja Portal for ArcGIS. Ahora, podrá acceder a su portal a través de la dirección URL de ArcGIS Web Adaptor, en lugar del puerto 7443. La dirección URL tiene el formato siguiente: https://webadaptorhost.domain.com/webadaptorname/home.

Si va a configurar Portal for ArcGIS con el servidor proxy de su organización, tendrá que proporcionar alguna información al portal acerca del servidor proxy. Para obtener instrucciones completas, consulte Configurar un servidor proxy con el portal.
