Habilitar automáticamente los componentes de IIS

Para instalar ArcGIS Web Adaptor se necesitan algunos componentes de Microsoft Internet Information Services (IIS) 7.0, 7.5, 8.0 u 8.5. Si ya ha habilitado IIS 7.0, 7.5, 8.0 u 8.5 pero faltan los componentes requeridos de IIS, el proceso de instalación mostrará un mensaje indicando que faltan ciertos componentes de IIS.

Tiene la opción de permitir que la instalación de Web Adaptor habilite automáticamente los componentes de IIS requeridos. Pero también puede habilitar los componentes por su cuenta. No se podrá proceder con la instalación si los componentes de IIS obligatorios no están habilitados.

Para habilitar manual o automáticamente los componentes de IIS que faltan, siga las instrucciones de las secciones siguientes.

Habilitar automáticamente los componentes de IIS que faltan

  1. Revise los componentes ausentes en el cuadro de diálogo Verificación de los requisitos de IIS. Tal vez le resulte útil anotar los componentes ausentes como recordatorio.
  2. Para habilitar automáticamente los componentes ausentes, haga clic en Acepto.

Los componentes se habilitan y se puede continuar el proceso de instalación.

Habilitar manualmente los componentes de IIS que faltan

Si desea habilitar manualmente los componentes de IIS 7.0, 7.5, 8.0 u 8.5 que faltan, haga clic en Cancelar. Para obtener más información sobre cómo puede habilitar manualmente los componentes que faltan de IIS por su cuenta, consulte los temas siguientes.

  • Habilitar IIS y los componentes obligatorios de IIS en Windows Server 2012
  • Habilitar IIS y los componentes obligatorios de IIS en Windows Server 2008/2008 R2
  • Habilitar IIS y los componentes requeridos de IIS en Windows 8/8.1
  • Habilitar IIS y los componentes obligatorios de IIS en Windows 7

Después de habilitar todos los componentes necesarios, vuelva a ejecutar la instalación de Web Adaptor haciendo doble clic en el archivo Setup.exe que reside en la carpeta de instalación de ArcGIS Web Adaptor (IIS).
