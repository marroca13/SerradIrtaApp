Instalar ArcGIS Web Adaptor de forma automática

Es posible instalar ArcGIS Web Adaptor ejecutando la configuración con los parámetros de la línea de comandos de Windows Installer, como se describe a continuación. También puede instalarlo desde la interfaz de usuario, utilizando Setup.exe.

Requisitos de instalación

La versión de ArcGIS Web Adaptor debe coincidir siempre con la versión de ArcGIS Server que esté usando.

Puede tener instalada la versión 10.2.1 o una versión posterior de ArcGIS Web Adaptor junto con una versión anterior de ArcGIS Web Adaptor en el mismo equipo. Por ejemplo, puede tener instalada la versión 10.1 SP1 y una versión 10.6 de ArcGIS Web Adaptor en el mismo servidor web.

Cada Web Adaptor debe tener su propio nombre único; no puede tener dos Web Adaptors con el mismo nombre en un único servidor web. Si se detecta un conflicto de nombres durante la instalación, se muestra un mensaje de advertencia. Debe desinstalar la versión anterior que tiene el mismo nombre para resolver el conflicto. También puede cambiar el nombre de un Web Adaptor para continuar con la instalación.

Se requiere Microsoft Internet Information Services (IIS). Solamente en el caso de instalaciones automáticas, si no cuenta con IIS, no se podrá proceder con la instalación de ArcGIS Web Adaptor. Antes de instalar ArcGIS Web Adaptor de modo automático, es necesario que IIS y todos los componentes necesarios estén instalados.

Se requiere Microsoft .NET Framework 4.5 o una versión superior. Si no se encuentra Microsoft .NET Framework 4.5. o una versión superior, no se podrá continuar con la instalación.

Legado:

En versiones anteriores, para poder instalar ArcGIS Web Adaptor se necesitaba Microsoft .NET Framework 3.5 Service Pack 1 (SP1), 4.0 o 4.5.

Una vez instalado ArcGIS Web Adaptor, se debe configurar. Para obtener más información, consulte Configurar ArcGIS Web Adaptor.

Actualizar una versión anterior de ArcGIS Web Adaptor

Para actualizar una versión anterior de ArcGIS Web Adaptor a 10.6, tiene que desinstalar la versión antigua e instalar la configuración de 10.6. Durante la instalación de 10.6, deberá especificar el mismo nombre del Web Adaptor anterior. Esto garantiza que la dirección URL de Web Adaptor sea idéntica a la de la versión anterior.

Si se detecta un conflicto de nombres durante la instalación de 10.6, se muestra un mensaje de advertencia. Debe desinstalar la versión anterior que tiene el mismo nombre para realizar la actualización.

Después de instalar 10.6, debe configurar ArcGIS Web Adaptor con su correspondiente sitio de ArcGIS Server.

Legado:

En la versión 10.2 y versiones anteriores, al ejecutar la configuración se actualizaban todas las versiones existentes de Web Adaptor a la versión más reciente.

Parámetros de línea de comandos para especificar el sitio Web y el nombre para ArcGIS Web Adaptor

Las siguientes líneas de comandos se utilizan para configurar IIS. Si no se invocan estos parámetros de línea de comandos durante una instalación automática, la aplicación Web de ArcGIS Web Adaptor se creará bajo el sitio Web predeterminado con el nombre predeterminado arcgis.

  • WEBSITE_ID=<valor numérico del sitio web>

    El parámetro WEBSITE_ID se utiliza para especificar el sitio web donde se creará la aplicación web de ArcGIS Web Adaptor. El valor para WEBSITE_ID es un número específico para cada sitio web en el servidor y lo asigna IIS. Los Id. para los sitios web en el servidor se pueden encontrar en el Administrador de IIS. El sitio web predeterminado tiene un Id. de sitio web igual a 1. Por defecto, el parámetro WEBSITE_ID se establece en el sitio web predeterminado (incluso si existen varias páginas Web). El parámetro WEBSITE_ID distingue entre mayúsculas y minúsculas.

  • PORT=<número de puerto>

    Además, si cuenta con un sitio web con muchos puertos, puede usar el parámetro PORT para instalar en el puerto especificado de WEBSITE_ID.

    Por ejemplo, la ruta es <path to ArcGIS Web Adaptor (IIS) setup download>\setup.exe /qb VDIRNAME=arcgis_external WEBSITE_ID=4059640 PORT=82.

  • VDIRNAME=<nombre de ArcGIS Web Adaptor>

    Este parámetro de línea de comando es opcional. Por defecto, la propiedad VDIRNAME está configurada como arcgis. Si no invoca el parámetro VDIRNAME en la línea de comandos, Web Adaptor se crea como arcgis. El parámetro VDIRNAME distingue entre mayúsculas y minúsculas. El nombre del Web Adaptor no puede contener espacios.

    Por ejemplo, la ruta es <path to ArcGIS Web Adaptor (IIS) setup download>\setup.exe /qb VDIRNAME=arcgis_external WEBSITE_ID=4059640.

  • CONFIGUREIIS=TRUE

    Este parámetro opcional permite al programa de instalación configurar e instalar de forma silenciosa todos los componentes de Microsoft IIS requeridos para ArcGIS Web Adaptor.
