Dar de baja ArcGIS Web Adaptor con Portal for ArcGIS

Si desea modificar Portal for ArcGIS para utilizar un ArcGIS Web Adaptor diferente, por ejemplo, si desea cambiar el nombre del Web Adaptor o el nombre del equipo que aloja ArcGIS Web Adaptor, primero tendrá que dar de baja ArcGIS Web Adaptor con el portal.

Deberá utilizar el Directorio de Portal for ArcGIS para dar de baja ArcGIS Web Adaptor con su portal. Solo puede darlo de baja a través del directorio de Portal for ArcGIS; la funcionalidad de dar de baja no está disponible desde la página de configuración de ArcGIS Web Adaptor ni desde el sitio web del portal.

Para dar de baja ArcGIS Web Adaptor, siga los pasos que se indican a continuación.

Pasos:
  1. Abra el Directorio de ArcGIS Portal e inicie sesión como Administrador de su organización. La URL tiene el formato https://portal.domain.com:7443/arcgis/portaladmin.
  2. Haga clic en Sistema > Web Adaptors.
  3. En la página Web Adaptors, haga clic en el nombre del equipo en el que esté alojado ArcGIS Web Adaptor.
  4. En la página Web Adaptor: <machine name>, haga clic en Anular registro de Web Adaptor.
  5. En la página Anular registro de Web Adaptor, haga clic en Anular registro de Web Adaptor. ArcGIS Web Adaptor se da de baja con el portal.
  6. Configure el nuevo ArcGIS Web Adaptor con el portal. Para ver las instrucciones completas, consulte Configurar ArcGIS Web Adaptor.
Nota:

Si su intención es eliminar por completo el anterior ArcGIS Web Adaptor del portal, se recomienda desinstalar ArcGIS Web Adaptor después de seguir los pasos anteriores. La desinstalación elimina todos los detalles de configuración de ArcGIS Web Adaptor del portal y bloquea el acceso al portal de los miembros y clientes mediante la URL de ArcGIS Web Adaptor dada de baja. Para ver las instrucciones completas, consulte Desinstalación de ArcGIS Web Adaptor.
