Requisitos del sistema para ArcGIS Web Adaptor 10.6

A continuación se indican los requisitos de sistema y de hardware para ejecutar ArcGIS Web Adaptor. Para conocer los requisitos adicionales y actualizados, consulte la página Requisitos del sistema disponible en la Web. Para obtener información sobre versiones anteriores, vea Esri Support.

Se recomienda consultar el aviso de obsolescencia para determinar si sus componentes de hardware y software siguen siendo compatibles con la versión 10.6.

Nota:

Windows 7, 8.1 y 10 solo se pueden utilizar para el desarrollo de aplicaciones y pruebas básicas. No se recomiendan para la implementación en un entorno de producción.

Número máximo de instancias de instalación

Puede instalar un máximo de 21 instancias de ArcGIS Web Adaptor en un solo equipo. Si necesita más instancias, tendrá que instalarlas en un equipo independiente.

Si el equipo tiene versiones anteriores instaladas, no es preciso desinstalarlas. Por ejemplo, puede tener 5 instancias de of ArcGIS Web Adaptor 10.4 y 21 instancias de ArcGIS Web Adaptor 10.6 instaladas en el mismo equipo. El valor máximo solo se aplica a las instancias de la misma versión de software.

Compatibilidad con Microsoft .NET Framework 4.5

Para instalar ArcGIS Web Adaptor se requiere Microsoft .NET Framework 4.5. o superior. Si su sistema operativo incluye Microsoft .NET Framework 4.5, asegúrese de que está habilitado en Características de Windows. Si su sistema operativo no incluye Microsoft .NET Framework 4.5, puede descargarlo desde el soporte de instalación y habilitarlo usando Características de Windows.

Componentes obligatorios de Microsoft IIS

ArcGIS Web Adaptor admite IIS en Windows 7, 8.1, 10, Server 2008 R2, Server 2012 y 2012 R2. Esto corresponde a las versiones de IIS 7.0, 7.5, 8.0, 8.5 y 10. IIS debe estar habilitado junto con componentes específicos de IIS 7.0, 7.5, 8.0, 8.5 y 10. No se podrá proceder con la instalación si no se detecta IIS y los componentes de IIS específicos no están habilitados.

Si IIS 7.0, 7.5, 8.0 u 8.5 ya está instalado pero faltan componentes requeridos de IIS, el programa de instalación mostrará el cuadro de diálogo Verificación de requisitos de IIS. Esto le brinda la opción de permitir que la instalación habilite automáticamente cualquier componente de IIS obligatorio que esté ausente. Para hacer esto, haga clic en Acepto. Para obtener más información, consulte Habilitar automáticamente los componentes de IIS.

Nota:

Si va a realizar una instalación automática de ArcGIS Web Adaptor, es necesario que todos los componentes de IIS obligatorios se habiliten manualmente. El programa de instalación no habilitará automáticamente los componentes que faltan de IIS mientras se está ejecutando una instalación automática.

Para obtener más información sobre cómo habilitar IIS en su sistema operativo y habilitar de forma opcional los componentes que faltan de IIS 7.0, 7.5, 8.0 u 8.5 por su cuenta, consulte los siguientes temas:
