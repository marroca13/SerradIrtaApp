Información general sobre cómo importar ArcGIS Web Adaptor con Portal for ArcGIS

A continuación se muestra una vista general de los pasos que es preciso seguir para implementar ArcGIS Web Adaptor con Portal for ArcGIS:

Pasos:
  1. Planifique cómo se va a usar ArcGIS Web Adaptor con Portal for ArcGIS.
  2. Verifique que el sistema cumpla con los requisitos mínimos del sistema.
  3. Habilite HTTPS en el servidor web.
  4. Instale ArcGIS Web Adaptor.
  5. Configurar ArcGIS Web Adaptor para usar con Portal for ArcGIS.
