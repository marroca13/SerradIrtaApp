Habilitar IIS y los componentes obligatorios de IIS en Windows Server 2008/2008 R2

ArcGIS Web Adaptor necesita que IIS esté habilitado y que determinados componentes de ISS estén habilitados en Windows Server 2008/2008 R2. No se podrá proceder con la instalación si no se detecta IIS y los componentes de IIS específicos no están habilitados.

Si ya ha habilitado IIS pero faltan los componentes necesarios de IIS, el proceso de instalación mostrará un mensaje indicando que faltan ciertos componentes de IIS. Tiene la opción de permitir que la instalación habilite automáticamente los componentes de IIS obligatorios. Sin embargo, dependiendo de las políticas de seguridad de su organización, tal vez sea necesario habilitar manualmente los componentes requeridos de IIS, tal y como se describe a continuación.

Para aprender a habilitar IIS y los componentes de IIS necesarios en Windows Server 2008/2008 R2, consulte las instrucciones siguientes.

  1. Haga clic en Inicio > Herramientas administrativas > Administrador del servidor.

  2. En el panel izquierdo del cuadro de diálogo Server Manager, haga clic en Roles.
    • Si IIS no se ha habilitado, haga clic en Agregar roles en el panel Resumen de roles. Haga clic en Siguiente y habilite Web Server (IIS) en la lista. A continuación, haga clic en Siguiente y seleccione Servicios de rol en el panel de la izquierda.
    • Si IIS ya está habilitado pero no se han habilitado todos los componentes requeridos, haga clic en Agregar servicios de rol en el panel Web Server (IIS) de la derecha.

  3. En el cuadro de diálogo Seleccionar servicios de rol, verifique que los componentes del servidor web que se enumeran a continuación estén habilitados.

  4. Después de habilitar los componentes necesarios de IIS, haga clic en Siguiente > Instalar.

Componentes de IIS necesarios

Los componentes de IIS que se detallan a continuación cumplen con los requisitos mínimos para ejecutar Web Adaptor. Si están habilitados otros componentes de IIS, no es necesario quitarlos.
