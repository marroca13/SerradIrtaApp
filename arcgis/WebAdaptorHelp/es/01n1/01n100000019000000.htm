Instalar ArcGIS Web Adaptor

ArcGIS Web Adaptor se suministra como un programa de instalación y se ejecuta mediante una utilidad denominada Setup.exe. Puede instalar ArcGIS Web Adaptor de forma independiente siguiendo los pasos que se indican aquí o puede instalarlo de forma automática utilizando los parámetros de la línea de comandos de Windows Installer.

Requisitos de instalación

La versión de ArcGIS Web Adaptor debe coincidir siempre con la versión de Portal for ArcGIS que esté usando.

ArcGIS Web Adaptor se puede instalar en paralelo con otras versiones. No es necesario desinstalar ArcGIS Web Adaptor para instalar ArcGIS Web Adaptor y ArcGIS Web Adaptor no es una actualización de versiones anteriores.

Cada ArcGIS Web Adaptor debe tener su propio nombre único; no puede tener dos Web Adaptors con el mismo nombre en un único servidor web. Si se detecta un conflicto de nombres durante la instalación, se muestra un mensaje de advertencia. Debe desinstalar la versión anterior que tiene el mismo nombre para resolver el conflicto. También puede cambiar el nombre de uno de los Web Adaptor para continuar con la instalación.

Actualizar una versión anterior de ArcGIS Web Adaptor

Para actualizar una versión anterior de ArcGIS Web Adaptor a 10.6, tiene que desinstalar la versión antigua e instalar la configuración de 10.6. Durante la instalación de 10.6, deberá especificar el mismo nombre del Web Adaptor anterior. Esto garantiza que la dirección URL de Web Adaptor sea idéntica a la de la versión anterior.

Si se detecta un conflicto de nombres durante la instalación de 10.6, se muestra un mensaje de advertencia. Debe desinstalar la versión anterior que tiene el mismo nombre para realizar la actualización.

Después de instalar 10.6, deberá configurar ArcGIS Web Adaptor con Portal for ArcGIS.

Legado:

En la versión 10.2 y versiones anteriores, al ejecutar la configuración se actualizaban todas las versiones existentes de Web Adaptor a la versión más reciente.

Antes de instalar ArcGIS Web Adaptor

Pasos:
  1. Verifique que el sistema cumpla todos los requisitos del sistema. Solo puede configurar un Web Adaptor con Portal for ArcGIS.
  2. Inicie sesión como usuario con privilegios administrativos.
  3. Compruebe que tenga un sitio web funcionando en el puerto 80 y HTTPS habilitado en el puerto 443. Esto sirve para acomodar las distintas llamadas encriptadas y desencriptadas que se realizan a Portal for ArcGIS. ArcGIS Web Adaptor no se puede configurar con Portal for ArcGIS si está instalado en un sitio web que se ejecuta en puertos distintos del 80 y 443. Puede consultar las instrucciones para crear un sitio web en la documentación de producto de IIS. Para obtener más información sobre la configuración y el uso de HTTPS, consulte Activar HTTPS en su servidor web.
  4. Cierre todas las aplicaciones.
  5. Examine la carpeta descargada que contiene el programa de instalación de ArcGIS Web Adaptor. También puede insertar el soporte de Portal for ArcGIS en la unidad correspondiente para iniciar automáticamente el programa de instalación.

Ahora está listo para proceder con la instalación de Web Adaptor.

Precaución:

No se puede intentar configurar ArcGIS Web Adaptor de modo que la dirección URL incluya niveles adicionales, y hacerlo puede causar problemas con las aplicaciones cliente.

Instalación de ArcGIS Web Adaptor

Pasos:
  1. El programa de instalación de ArcGIS Web Adaptor debería iniciarse automáticamente después de completar la descarga. Si el programa de instalación no se inicia automáticamente, vaya a la ubicación de descarga de los archivos de instalación y haga doble clic en Setup.exe.
  2. Revise los términos y condiciones del acuerdo maestro. Debe aceptar los términos para continuar.
  3. Elija un sitio web que se ejecute en el puerto 80 de ArcGIS Web Adaptor. Los sitios Web disponibles se enumeran como <nombre del sitio web (puerto)>. Si no puede ver el cuadro de diálogo Seleccionar sitio web, significa que sólo tiene un sitio web. Si solo hay un sitio web en el equipo, ArcGIS Web Adaptor se coloca automáticamente en el sitio web sin mostrar el cuadro de diálogo Seleccionar sitio web.
  4. Proporcione un nombre para Web Adaptor. El valor predeterminado es arcgis. El nombre de instancia no puede contener espacios.
    Nota:

    Aparecerá un mensaje si ya existe un directorio virtual con el mismo nombre que Web Adaptor en el sitio web seleccionado. Haga clic en Aceptar para cerrar el mensaje y proporcione un nombre diferente para Web Adaptor.

  5. Para terminar la instalación, siga las instrucciones en pantalla.

La página de configuración de Web Adaptor aparece después de finalizar la instalación. Ahora ya está preparado para Configurar ArcGIS Web Adaptor y utilizarlo en el portal.
