Configurar el idioma de visualización para ArcGIS Web Adaptor

El programa de instalación de ArcGIS Web Adaptor incluye versiones de Web Adaptor previamente traducidas a distintos idiomas. Estos idiomas previamente localizados son los siguientes:

  • Árabe
  • Chino simplificado
  • Francés
  • Alemán
  • Italiano
  • Japonés
  • Portugués brasileño
  • Ruso
  • Español
  • Turco

Cuando la instalación se ha completado, la página de configuración de Web Adaptor se abre automáticamente. Si el navegador ya está configurado para mostrar uno de los idiomas antes mencionados, la aplicación de Web Adaptor se abre automáticamente en ese idioma.

Para visualizar Web Adaptor en otro idioma, configure el navegador o sistema tal como se describe a continuación.

Usuarios de Firefox

Versiones localizadas de Firefox están disponibles. Al instalar una versión traducida de Firefox y abrir Web Adaptor, se visualizará automáticamente en el mismo idioma que el navegador.

Si desea utilizar Firefox y visualizar Web Adaptor en un idioma distinto del idioma del navegador, debe utilizar Firefox 10 o una versión posterior y configurar lo siguiente:

  1. Seleccione Herramientas > Opciones.
  2. En la pestaña Contenido, en Idiomas, haga clic en Elegir.
  3. Seleccione el idioma que desea mostrar y muévalo al inicio de la lista de idiomas.

Usuarios de Chrome

Si utiliza Google Chrome y desea modificar el idioma en el que se muestra en la aplicación de Web Adaptor, configure lo siguiente:

  1. Elija Personalizar > Opciones > Opciones avanzadas.
  2. En la sección Contenido Web, haga clic en Opciones de idioma y corrector ortográfico.
  3. Seleccionar el idioma que desea mostrar y, a continuación, haga clic en Mostrar Google Chrome en este idioma.

Usuarios de Internet Explorer

Si utiliza Internet Explorer y desea modificar el idioma en el que se muestra en Web Adaptor, configure lo siguiente:

  1. Elija Herramientas > Opciones de Internet.
  2. En la pestaña General, bajo Apariencia, haga clic en el botón Idiomas.
  3. Seleccione el idioma que desea mostrar y muévalo al inicio de la lista de idiomas. Haga clic en Aceptar y, a continuación, haga clic en Aceptar de nuevo para aceptar el cambio.
