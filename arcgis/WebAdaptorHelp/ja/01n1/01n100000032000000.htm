My Esri

My Esri は、Esri と対話しながら、質疑応答、フィードバックの投稿、情報の入手などを行うための Web サイトです。この Web サイトには、よくあるご質問、操作方法、ソフトウェア ダウンロード、トラブルシューティングのヒントなども含まれています。Esri と連絡を取るには、Esri アカウントで Web サイトにログインします。または、カスタマー サービスまたは地域の販売代理店に連絡することもできます。

Web ヘルプ

ArcGIS のソフトウェアおよびサービスに関する最新情報については、ArcGIS Desktop、ArcGIS Enterprise、および ArcGIS for Developers の Web サイトにアクセスしてください。これらは、GIS テクノロジについての理解を深めるのに役立つ非常に重要なリソースです。

インストールされたヘルプ

ArcGIS をインストールすると、ArcGIS ソフトウェアのヘルプがインストールされます。Windows オペレーティング システムのコンピューターで、ヘルプへのショートカットを利用するには、[スタート] → [すべてのプログラム] → [ArcGIS] の順に進みます。目次を調べたり、索引で用語を検索したり、セクションの中で何度も登場する用語を調べることで、必要な情報を見つけることができます。
