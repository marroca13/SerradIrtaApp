複数の ArcGIS Web Adaptor のインストール

同じコンピューター上に複数の ArcGIS Web Adaptor を同時にインストールできます。インストールは、別のコンピューターで繰り返し実行することもできます。プロセスの手順は以下のとおりです。

複数の ArcGIS Web Adaptor のインストール方法

複数の ArcGIS Web Adaptor は、同じコンピューター上に同時にインストールしたり、複数のコンピューターにインストールしたりできます。複数の ArcGIS Web Adaptor を同じ Web サイト (ポート) にインストールする場合、別々の名前を使用する必要があります。たとえば、arcgis という名前の 2 つの ArcGIS Web Adaptor を同じ Web サイトに配置することはできません。複数の ArcGIS Web Adaptor を同じ名前にしたい場合は、異なる Web サイト (ポート) 上にインストールする必要があります。

  1. ArcGIS Web Adaptor (IIS) のセットアップをダウンロードしたフォルダーに移動します。または、ArcGIS Server メディアを適切なドライブに挿入すると、メディアのフロント エンドが自動的に起動します。
  2. 最初に ArcGIS Web Adaptor をインストールしたときと同じ手順に従います。詳細については、「ArcGIS Web Adaptor のインストール」をご参照ください。
注意:

セットアップは、ArcGIS Web Adaptor の名前が競合していないかを検出して、競合を防ぎます。仮想ディレクトリがすでに存在するというメッセージが表示される場合は、ArcGIS Web Adaptor に別の名前を指定するか、別の Web サイトを選択します。

注意:

ArcGIS Web Adaptor サービス パックを適用するためにベース インスタンスを削除することはできません。

複数の ArcGIS Web Adaptor をサイレント インストールする手順

Windows インストーラー コマンド ライン パラメーターを使用してセットアップを実行すると、複数の ArcGIS Web Adaptor をインストールできます。コマンド ライン パラメーターの使用方法の詳細手順については、「ArcGIS Web Adaptor のサイレント インストール」をご参照ください。複数の ArcGIS Web Adaptor をインストールするのに、追加のコマンド ライン パラメーターは必要ありません。

複数の ArcGIS Web Adaptor をアンインストールする手順

各 ArcGIS Web Adaptor は別々にアンインストールする必要があります。手順の詳細については、「ArcGIS Web Adaptor のアンインストール」をご参照ください。
