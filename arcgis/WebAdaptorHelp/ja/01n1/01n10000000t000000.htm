ArcGIS Web Adaptor への ArcGIS Server の実装の概要

ArcGIS Server に ArcGIS Web Adaptor を実装する際に実行する手順の概要を次に示します。

  1. ArcGIS Server のサイト構成を計画します。
  2. システムが最小システム要件を満たしていることを確認します。
  3. ArcGIS Web Adaptor をインストールします。
  4. ArcGIS Server と組み合わせて使用できるように ArcGIS Web Adaptor を構成します。
