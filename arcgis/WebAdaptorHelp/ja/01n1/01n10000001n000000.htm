ArcGIS Web Adaptor の概要

ArcGIS Web Adaptor は、ArcGIS Server または Portal for ArcGIS と連携するように構成できます。1 つの ArcGIS Web Adaptor アプリケーションを両方と連携するように構成することは、技術的には可能ですが、サポートされていません。ArcGIS Web Adaptor の使い方が決まったら、以下のセクションをご参照ください。
