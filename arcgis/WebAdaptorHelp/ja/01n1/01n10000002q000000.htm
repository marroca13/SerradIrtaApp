Portal for ArcGIS からの ArcGIS Web Adaptor の登録解除

Portal for ArcGIS を変更して、別の ArcGIS Web Adaptor を使用する場合 (たとえば、ArcGIS Web Adaptor の名前または ArcGIS Web Adaptor をホストするコンピューターの名前を変更する場合)、はじめにポータルから ArcGIS Web Adaptor を登録解除する必要があります。

ポータルから ArcGIS Web Adaptor を登録解除するには、ArcGIS Portal Directory を使用します。登録解除する方法は ArcGIS Portal Directory の使用以外にはありません。登録解除機能は、ArcGIS Web Adaptor の構成ページまたはポータル Web サイトからは使用できません。

ArcGIS Web Adaptor を登録解除するには、以下の手順を実行します。

手順:
  1. ArcGIS Portal Directory を開いて、組織の管理者としてサイン インします。URL の形式は、https://portal.domain.com:7443/arcgis/portaladmin です。
  2. [System] → [Web Adaptors] の順にクリックします。
  3. [Web Adaptors] ページで、ArcGIS Web Adaptor をホストするコンピューターの名前をクリックします。
  4. [Web Adaptor: <コンピューター名>] ページで、[Unregister Web Adaptor] をクリックします。
  5. [Unregister Web Adaptor] ページで、[Unregister Web Adaptor] をクリックします。ArcGIS Web Adaptor がポータルから登録解除されます。
  6. ポータルに新しい ArcGIS Web Adaptor を構成します。詳細については、「ArcGIS Web Adaptor の構成」をご参照ください。
注意:

以前の ArcGIS Web Adaptor をポータルから完全に削除する場合は、上記の手順に従ってから ArcGIS Web Adaptor をアンインストールすることをお勧めします。アンインストールすると、ArcGIS Web Adaptor の構成の詳細がポータルからすべて削除され、メンバーとクライアントは、登録解除された ArcGIS Web Adaptor の URL を通じてポータルにアクセスできなくなります。手順の詳細については、「ArcGIS Web Adaptor のアンインストール」をご参照ください。
