ArcGIS Server での ArcGIS Web Adaptor の登録解除

ArcGIS Server を変更して、別の ArcGIS Web Adaptor を使用する場合 (たとえば、ArcGIS Web Adaptor の名前または ArcGIS Web Adaptor をホストするコンピューターの名前を変更する場合)、はじめに ArcGIS Server サイトから ArcGIS Web Adaptor を登録解除する必要があります。

ArcGIS Server Administrator Directory を使用して、ArcGIS Server サイトから ArcGIS Web Adaptor を登録解除します。登録解除する方法は ArcGIS Server Administrator Directory の使用以外にはありません。登録解除機能は、ArcGIS Web Adaptor の構成ページからは使用できません。

ArcGIS Web Adaptor の登録を解除するには、以下の手順に従います。

手順:
  1. ArcGIS Server Administrator Directory を開いて、サイトの管理者としてサイン インします。URL の形式は、https://gisserver.domain.com:6443/arcgis/admin です。
  2. [System] → [Web Adaptors] の順にクリックします。
  3. [Web Adaptors] ページで、ArcGIS Web Adaptor を表す文字列をクリックします。
  4. [Web Adaptor] ページで、リストされた URL が、登録解除する ArcGIS Web Adaptor を指していることを確認します。[Unregister] をクリックします。
  5. [Unregister Web Adaptor] ページで、[Unregister Web Adaptor] をクリックします。ArcGIS Web Adaptor が ArcGIS Server サイトから登録解除されます。
  6. ArcGIS Server サイトで新しい ArcGIS Web Adaptor を構成します。詳細な手順については、「ArcGIS Web Adaptor の構成」をご参照ください。
注意:

以前の ArcGIS Web Adaptor を ArcGIS Server サイトから完全に削除する場合は、上記の手順に従ってから ArcGIS Web Adaptor をアンインストールすることをお勧めします。アンインストールすると、ArcGIS Web Adaptor の構成の詳細がサイトからすべて削除され、メンバーとクライアントは、登録解除された ArcGIS Web Adaptor の URL を通じてサイトにアクセスできなくなります。手順の詳細については、「ArcGIS Web Adaptor のアンインストール」をご参照ください。
