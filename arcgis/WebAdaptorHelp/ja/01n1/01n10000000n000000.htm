ArcGIS Web Adaptor のアンインストール

ArcGIS Web Adaptor をアンインストールすると、ArcGIS Web Adaptor は ArcGIS Server から登録解除され、クライアント アプリケーションはサーバーと通信できなくなります。後で通信を再確立するには、ArcGIS Web Adaptor を再インストールし、構成ページを使用して、ArcGIS Web Adaptor をサーバーに再構成します。手順の詳細については、「ArcGIS Web Adaptor の構成」をご参照ください。

ArcGIS Web Adaptor をアンインストールするには、下記のセクションの手順に従います。

ArcGIS Web Adaptor のアンインストール

  1. Windows 7 または Windows Server 2008/R2 の場合、[スタート] → [コントロール パネル] の順にクリックしてから (必要に応じてクラシック表示に切り替える)、[プログラムと機能] を選択します。Windows 8/8.1 または Windows Server 2012/R2 の場合、[コントロール パネル] を開いて、[プログラムと機能] をクリックします。
  2. [プログラム] リストから [ArcGIS Web Adaptor (IIS)] を選択します。
  3. [アンインストールと変更] を選択して、コンピューターから ArcGIS Web Adaptor をアンインストールします。
