ArcGIS Web Adaptor の表示言語の設定

ArcGIS Web Adaptor セットアップには、複数の言語にローカライズされた ArcGIS Web Adaptor が含まれています。このローカライズ版の言語には、以下のようなものがあります。

  • アラビア語
  • 簡体字中国語
  • フランス語
  • ドイツ語
  • イタリア語
  • 日本語
  • ブラジル ポルトガル語
  • ロシア語
  • スペイン語
  • トルコ語

セットアップが完了すると、ArcGIS Web Adaptor 構成ページが自動的に開きます。ブラウザーが、上記の言語のいずれかを使用して表示されるように構成されている場合、ArcGIS Web Adaptor アプリケーションはその言語で自動的に開きます。

別の言語で ArcGIS Web Adaptor を開くには、ブラウザーまたはシステムを以下の説明のとおりに構成します。

Firefox ユーザー

Firefox は、ローカライズ版が提供されています。Firefox のローカライズ版をインストールして、Web Adaptor を開くと、自動的にブラウザーの言語でWeb Adaptor が表示されます。

Firefox をご使用の場合に、ArcGIS Web Adaptor をブラウザーの言語とは別の言語で表示するには、Firefox 10 以降を使用したうえで、次のように構成する必要があります。

  1. [ツール] → [オプション] の順に選択します。
  2. [コンテンツ] タブの [言語] の下の [言語設定] をクリックします。
  3. 表示に使用する言語を選択して、その言語を言語リストの一番上に移動します。

Chrome をご使用の場合

Google Chrome を使用している場合に、ArcGIS Web Adaptor アプリケーションの表示言語を変更するには、次のように構成します。

  1. [Google Chromeの設定] → [オプション] → [高度な設定] の順に選択します。
  2. [言語] セクションで、[言語とスペルチェックの設定] をクリックします。
  3. 表示に使用する言語を選択して、[GoogleChrome をこの言語で表示] をクリックします。

Internet Explorer をご使用の場合

Internet Explorer を使用している場合に、ArcGIS Web Adaptor の表示言語を変更するには、次のように構成します。

  1. [ツール] → [インターネット オプション] の順に選択します。
  2. [全般] タブの [表示設定] の下の [言語] ボタンをクリックします。
  3. 表示に使用する言語を選択して、その言語を言語リストの一番上に移動します。[OK] をクリックしてから、もう一度 [OK] をクリックして、変更を適用します。
