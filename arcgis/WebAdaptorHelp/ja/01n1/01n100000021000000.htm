Portal for ArcGIS への ArcGIS Web Adaptor の実装の概要

Portal for ArcGIS に ArcGIS Web Adaptor を実装する際に従う手順の概要は次のとおりです。

手順:
  1. ArcGIS Web Adaptor を Portal for ArcGIS と組み合わせて使用する方法を計画します。
  2. システムが最小システム要件を満たしていることを確認します。
  3. Web サーバーで HTTPS を有効化します。
  4. ArcGIS Web Adaptor をインストールします。
  5. Portal for ArcGIS と組み合わせて使用できるように ArcGIS Web Adaptor を構成します。
