Отмена регистрации ArcGIS Web Adaptor на Portal for ArcGIS

Если вы хотите изменить Portal for ArcGIS для использования другого ArcGIS Web Adaptor, например, если вы хотите изменить имя ArcGIS Web Adaptor или имя одного из компьютеров, на котором размещен ArcGIS Web Adaptor, то вам потребуется сначала отменить регистрацию ArcGIS Web Adaptor на вашем портале.

Вы будете использовать ArcGIS Portal Directory для отмены регистрации ArcGIS Web Adaptor на вашем портале. Вы можете отменить регистрацию только с помощью ArcGIS Portal Directory; функциональность отмены регистрации недоступна со страницы конфигурации ArcGIS Web Adaptor или веб-сайта портала.

Чтобы отменить регистрацию ArcGIS Web Adaptor, выполните следующие шаги.

Шаги:
  1. Откройте ArcGIS Portal Directory и войдите в качестве администратора вашей организации. Адрес URL в формате https://portal.domain.com:7443/arcgis/portaladmin.
  2. Щелкните Система > Web Adaptor.
  3. На странице Web Adaptors щелкните имя компьютера, на котором размещен ArcGIS Web Adaptor.
  4. На странице Web Adaptor: <имя компьютера> щелкните Отменить регистрацию Web Adaptor.
  5. На странице Отменить регистрацию Web Adaptor щелкните Отменить регистрацию Web Adaptor Регистрация ArcGIS Web Adaptor на вашем портале отменена.
  6. Настройте новый ArcGIS Web Adaptor на вашем портале. Подробные инструкции см. в разделе Настройка ArcGIS Web Adaptor.
Примечание:

Если вы хотите полностью удалить предыдущую версию ArcGIS Web Adaptor с портала, рекомендуется удалить его после выполнения следующих шагов. При удалении с портала будет стерта вся информация о настройке ArcGIS Web Adaptor, и доступ пользователей и клиентов к порталу через URL-адрес ArcGIS Web Adaptor, регистрация которого была отменена, будет заблокирован. Подробные инструкции см. в разделе Удаление ArcGIS Web Adaptor.
