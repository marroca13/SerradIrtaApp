Установка ArcGIS Web Adaptor

ArcGIS Web Adaptor поставляется в виде программы установки и запускается с помощью служебного приложения Setup.exe. Установку ArcGIS Web Adaptor можно производить, следуя приведенной здесь пошаговой инструкции, или в автоматическом режиме с помощью параметров командной строки Windows Installer.

Требования установки

Версия ArcGIS Web Adaptor должна всегда совпадать с версией используемого вами Portal for ArcGIS.

ArcGIS Web Adaptor может быть установлен бок о бок с другими версиями. Для установки ArcGIS Web Adaptor де-инсталляция ArcGIS Web Adaptor не требуется, а ArcGIS Web Adaptor не является обновлением предыдущих версий.

Каждый ArcGIS Web Adaptor должен иметь собственное уникальное имя; на одном веб-сервере не должно быть два Web Adaptor с одинаковыми именами. Если во время установки обнаружен конфликт имён, появляется предупреждение. Вам будет необходимо удалить более раннюю версию с тем же именем, чтобы разрешить этот конфликт. Вы также можете изменить имя одного из экземпляров Web Adaptor, чтобы продолжить процесс установки.

Обновите предыдущую версию ArcGIS Web Adaptor

Для обновления предыдущей версии ArcGIS Web Adaptor до 10.6, необходимо удалить предыдущую версию и запустить программу установки для версии 10.6. Во время установки 10.6 вы укажете такое же имя, как и у вашего предыдущего Web Adaptor. Это гарантирует, что URL-адрес вашего Web Adaptor останется таким же, какой он был у предыдущей версии.

Если во время установки 10.6 обнаружен конфликт имён, появляется предупреждение. Вам будет необходимо удалить раннюю версию с тем же именем, чтобы выполнить обновление.

После установки 10.6 настроить ArcGIS Web Adaptor с Portal for ArcGIS.

Прежние версии:

В версии 10.2 и более ранних версиях запуск программы установки обновлял все существующие экземпляры Web Adaptor до самой последней версии.

Перед установкой ArcGIS Web Adaptor

Шаги:
  1. Убедитесь в том, что ваш компьютер отвечает всем системным требованиям. Для работы с Portal for ArcGIS можно настроить только один Web Adaptor.
  2. Войдите в операционную систему в качестве пользователя с правами доступа администратора.
  3. Убедитесь, что ваш сайт запущен на порте 80, а HTTPS включен на порте 443. Это сделано для того, чтобы приспособить различные зашифрованные и незашифрованные запросы к Portal for ArcGIS. ArcGIS Web Adaptor нельзя настроить с Portal for ArcGIS, если он установлен на сайте, который работает на порту, отличном от 80 и 443. Инструкции о том, как построить сайт, можно найти в документации к вашему продукту IIS. Подробные сведения о настройке и использовании HTTPS см. в разделе Включение HTTPS на вашем веб-сервере.
  4. Закройте все работающие приложения.
  5. Перейдите к загруженной папке, содержащей программу установки ArcGIS Web Adaptor. Также можно установить диск Portal for ArcGIS в соответствующий привод, чтобы автоматически запустить программу установки.

Теперь вы готовы начать установку Web Adaptor.

Внимание:

Попытка настроить ArcGIS Web Adaptor таким образом, чтобы URL-адрес содержал дополнительные уровни, не поддерживается и может вызвать проблемы в работе клиентских приложений.

Порядок установки ArcGIS Web Adaptor

Шаги:
  1. Программа установки ArcGIS Web Adaptor должна запускаться автоматически после завершения загрузки. Если программа установки автоматически не запустилась, перейдите в папку, где находятся загруженные установочные файлы, и дважды щелкните Setup.exe.
  2. Ознакомьтесь с условиями основного соглашения. Для продолжения необходимо принять условия.
  3. Выберите веб-сайт для ArcGIS Web Adaptor, работающий на порту 80. Возможные веб-сайты перечислены в виде <название веб-сайта (порт)>. Если диалоговое окно Выберите веб-сайт не появилось, значит, доступен только один веб-сайт. Если на вашем компьютере будет найден только один веб-сайт, то на этом веб-сайте будет автоматически установлен ArcGIS Web Adaptor, а диалоговое окно Выбрать веб-сайт не будет показано.
  4. Введите имя для Web Adaptor. По умолчанию задано название arcgis. Имя экземпляра не может содержать пробелов.
    Примечание:

    Если на веб-сайте уже существует виртуальная директория с таким же именем, как и у Web Adaptor, появится соответствующее сообщение. Щелкните OK, чтобы закрыть сообщение и введите для Web Adaptor другое имя.

  5. Для завершения процесса установки следуйте указаниям на экране.

После завершения установки откроется страница настройки Web Adaptor. Теперь вы готовы к Настроить ArcGIS Web Adaptor для работы с вашим порталом.
