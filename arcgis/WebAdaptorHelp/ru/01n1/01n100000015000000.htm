Автоматическая активация компонентов IIS

Для установки ArcGIS Web Adaptor требуются некоторые компоненты Microsoft Internet Information Services (IIS) 7.0, 7.5, 8.0 или 8.5. Если вы уже включили IIS 7.0, 7.5, 8.0 или 8.5, но обязательные компоненты IIS отсутствуют, появится сообщение программы установки с указанием отсутствующих компонентов IIS.

Вы можете разрешить программе установки Web Adaptor автоматически включить отсутствующие компоненты IIS. Вы также можете включить их самостоятельно. Установка не будет продолжена, если обязательные компоненты IIS не включены.

Следуйте приведенным в следующих разделах инструкциям, чтобы автоматически или вручную включить отсутствующие компоненты IIS.

Автоматическое включение отсутствующих компонентов IIS

  1. Просмотрите отсутствующие компоненты IIS в диалоговом окне Проверка требований IIS (IIS requirements verification). Вы можете записать для себя отсутствующие компоненты.
  2. Чтобы автоматически включить отсутствующие компоненты, щелкните Я согласен (I Agree).

Компоненты будут включены, и вы сможете продолжить процесс установки.

Включение отсутствующих компонентов IIS вручную

Если вы хотите включить отсутствующие компоненты IIS 7.0, 7.5, 8.0 или 8.5 вручную, щелкните Отмена (Cancel). Чтобы узнать, как вручную активировать отсутствующие компоненты IIS, обратитесь к указанным ниже разделам.

  • Включение IIS и обязательных компонентов IIS в Windows Server 2012
  • Включение IIS и обязательных компонентов IIS в Windows Server 2008/2008 R2
  • Включение IIS и обязательных компонентов IIS в Windows 8/8.1
  • Включение IIS и обязательных компонентов IIS в Windows 7

Как только все обязательные компоненты будут включены, снова запустите процесс установки Web Adaptor, дважды щелкнув на файле Setup.exe, находящийся в папке программы установки ArcGIS Web Adaptor (IIS).
