Установка нескольких ArcGIS Web Adaptor

Можно одновременно установить несколько экземпляров ArcGIS Web Adaptor на один компьютер. Процесс установки может также повторяться на различных машинах. Для этого выполните следующие шаги.

Как установить несколько экземпляров Web Adaptor

Несколько экземпляров Web Adaptor можно установить на один компьютер или распределить их по нескольким. Если вы хотите установить несколько экземпляров Web Adaptor на один веб-сайт (порт), необходимо задать им разные имена. Например, два экземпляра Web Adaptor с именем arcgis не могут находиться на одном веб-сайте. Если вы хотите установить несколько экземпляров Web Adaptor с одинаковыми именами, необходимо установить их на разные веб-сайты (порты).

  1. Перейдите в загруженную папку, содержащую программу установки ArcGIS Web Adaptor (IIS), или вставьте установочный носитель ArcGIS Server в соответствующий дисковод для автоматического запуска программы установки.
  2. Следуйте тем же инструкциям, которые вы использовали для установки первого Web Adaptor. Подробнее см. раздел Установка ArcGIS Web Adaptor.
Примечание:

Программа установки найдет и предотвратит возникновение любых конфликтов, связанных с именем Web Adaptor. Если вы видите сообщение о том, что виртуальный каталог уже существует, введите другое имя экземпляра Web Adaptor или выберите другой веб-сайт.

Примечание:

Для того, чтобы применить пакет обновлений Web Adaptor, нельзя удалить основной экземпляр.

Как автоматически установить несколько экземпляров Web Adaptor

Несколько экземпляров Web Adaptor можно установить, запустив установку с использованием параметров командной строки Windows Installer. Инструкции по использованию параметров командной строки находятся в разделе Автоматическая установка ArcGIS Web Adaptor. Для установки нескольких экземпляров Web Adaptor не требуется никаких дополнительных параметров командной строки.

Как удалить установку нескольких экземпляров Web Adaptor

Каждый экземпляр Web Adaptor должен быть удален отдельно. Для получения подробной информации см. раздел Удаление ArcGIS Web Adaptor.
