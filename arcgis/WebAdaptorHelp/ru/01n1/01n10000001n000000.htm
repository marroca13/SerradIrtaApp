Начало работы с ArcGIS Web Adaptor

Вы можете настроить ArcGIS Web Adaptor для работы с ArcGIS Server или с Portal for ArcGIS. Настройка одного приложения ArcGIS Web Adaptor для работы с обоими технически возможна, но официально не поддерживается. Если вы решили работать с ArcGIS Web Adaptor, подробную информацию о том, как начать с ним работать, см. в следующих разделах.
