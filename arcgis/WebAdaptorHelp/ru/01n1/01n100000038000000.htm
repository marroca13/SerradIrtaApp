Отмена регистрации ArcGIS Web Adaptor с ArcGIS Server

Если вы хотите изменить ArcGIS Server для использования другого ArcGIS Web Adaptor, например, если вы хотите изменить имя Web Adaptor или имя одного из компьютеров, на котором размещен ArcGIS Web Adaptor, то потребуется сначала отменить регистрацию ArcGIS Web Adaptor на вашем сайте ArcGIS Server.

Вы воспользуетесь ArcGIS Server Administrator Directory для отмены регистрации ArcGIS Web Adaptor для вашего сайта ArcGIS Server. Вы можете отменить регистрацию только с помощью ArcGIS Server Administrator Directory; эта функциональность не доступна со страницы конфигурации ArcGIS Web Adaptor.

Выполните следующие шаги, чтобы отменить регистрацию ArcGIS Web Adaptor.

Шаги:
  1. Откройте ArcGIS Server Administrator Directory и войдите в качестве администратора на ваш сайт. Адрес URL в формате https://gisserver.domain.com:6443/arcgis/admin.
  2. Щелкните Система > Web Adaptor.
  3. На странице Web Adaptors щелкните строку, представляющую ArcGIS Web Adaptor.
  4. На странице Web Adaptor убедитесь, что указанный URL относится к тому Web Adaptor, регистрацию которого вы хотите отменить. Щелкните Отменить регистрацию.
  5. На странице Отменить регистрацию Web Adaptor щелкните Отменить регистрацию Web Adaptor. Регистрация ArcGIS Web Adaptor отменена для вашего сайта ArcGIS Server.
  6. Настройка нового ArcGIS Web Adaptor для вашего сайта ArcGIS Server. Подробные инструкции см. в разделе Настройка ArcGIS Web Adaptor.
Примечание:

Если вы хотите полностью удалить предыдущий ArcGIS Web Adaptor из вашего сайта ArcGIS Server, рекомендуется удалить ArcGIS Web Adaptor после выполнения указанных выше шагов. При удалении с сайта будет стерта вся информация о настройке ArcGIS Web Adaptor, и доступ пользователей и клиентов к сайту через URL-адрес ArcGIS Web Adaptor, регистрация которого была отменена, будет заблокирован. Подробные инструкции см. в разделе Удаление ArcGIS Web Adaptor.
